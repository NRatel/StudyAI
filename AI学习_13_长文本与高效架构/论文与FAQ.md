# 论文与 FAQ

这个模块是高级选读。建议先用“它优化什么成本”来理解论文，而不是直接陷入公式。

## 推荐了解的主题

| 主题 | 可以关注什么 |
|------|--------------|
| RoPE 扩展 | 位置编码如何适应更长上下文 |
| Longformer / Sparse Attention | 为什么不让所有 token 两两相看 |
| FlashAttention | 如何让标准注意力更高效 |
| MoE | 稀疏激活如何扩大容量 |
| Mamba / SSM | 长序列建模的另一类路线 |

## FAQ

### 长上下文越长越好吗？

不一定。更长意味着成本更高，也可能引入更多干扰。关键是能不能找到并使用关键信息。

### MoE 参数多是不是一定更强？

不一定。要看激活参数、训练数据、路由效果和推理成本。

### FlashAttention 会改变模型能力吗？

通常不改变注意力数学结果，主要是更高效地计算。

### Mamba 会完全替代 Transformer 吗？

不能简单这样看。不同架构适合不同场景，未来更可能是多种路线并存和融合。
