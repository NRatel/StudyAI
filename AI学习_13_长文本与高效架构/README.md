# 13 长文本与高效架构

这个模块是高级选读。它回答：当上下文更长、模型更大、成本更高时，架构怎样继续扩展。

## 先抓住这条线

```text
上下文变长 → 注意力更贵
模型变大 → 每次全量计算更贵
需要更省 → 长上下文、MoE、高效注意力、SSM 等路线出现
```
