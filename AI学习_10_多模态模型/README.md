# AI学习：10 多模态模型

> **内容时效**：截至 2026 年 4 月。多模态是当前 AI 最活跃的前沿方向之一，技术迭代极快，请结合最新资讯学习。

## 本模块导读

**学什么**：多模态模型——让 AI 同时理解和生成文本、图像、视频、语音等多种信息形态的技术体系。从 Vision Transformer（ViT）对图像建模方式的革命、到 CLIP 实现图文对齐的范式突破、到 GPT-4V/4o 等多模态大模型、再到 Sora 视频生成和 Whisper 语音理解，本模块覆盖多模态 AI 的核心原理与工程实践。

**为什么学**：人类感知世界本就是多模态的——我们同时通过视觉、听觉、语言来理解环境。2023-2026 年，多模态能力已成为衡量 AI 系统智能程度的核心维度。GPT-4o、Gemini、Claude 等最强模型无一不是多模态的。理解多模态技术，是理解当前 AI 发展方向的必修课。

**学完能做什么**：
- 理解 ViT 如何将图像转化为 Transformer 可处理的 token 序列
- 理解 CLIP 的对比学习范式，以及"图文对齐"为什么如此重要
- 理解 GPT-4V/4o、LLaVA 等多模态大模型的架构设计
- 理解视频理解的时空建模方法和 Sora 的技术原理
- 理解 Whisper 语音模型及语音与 LLM 的融合方案
- 具备对多模态系统进行技术选型和架构设计的能力

**前置知识**：
- [05 GPT系列深度解析](../AI学习_05_GPT系列深度解析/README.md)（LLM 架构、Scaling Laws、对齐技术）
- [06 生成模型与扩散模型](../AI学习_06_生成模型与扩散模型/README.md)（扩散模型原理、DiT 架构、Stable Diffusion）

**预估学习时间**：6~8 小时

**难度与定位**：★★★★，前沿篇

## 学习建议

按序号 1 → 3 顺序阅读，每个文件内部都是自包含的"直觉 → 原理 → 代码 → 工程视角"结构。

- **第 1 节**是多模态的基石——ViT 解决了"图像如何进入 Transformer"的问题，CLIP 解决了"图像和文本如何对齐"的问题。这两个基础不牢，后续内容无法理解。
- **第 2 节**是当前最实用的方向——GPT-4V/4o 和 LLaVA 代表了"视觉编码器 + LLM"的主流多模态架构范式。
- **第 3 节**是前沿拓展——视频和语音是多模态的自然延伸，Sora 和 Whisper 是各自领域的标志性工作。

建议动手实践：用 CLIP 做一个零样本图像分类器，用 LLaVA 或 GPT-4o API 做一个图文问答 Demo，体感远胜于纯理论。

## 多模态技术全景图（截至 2026.4）

```
多模态模型技术栈
│
├── 视觉编码器
│   ├── ViT（2020）→ DINOv2（2023）→ SigLIP（2023）→ InternViT（2024）
│   └── CLIP 视觉编码器（最广泛使用）
│
├── 图文对齐
│   ├── CLIP 对比学习（2021）
│   ├── SigLIP sigmoid 对比（2023）
│   └── Q-Former / Perceiver Resampler（跨模态桥接）
│
├── 多模态大模型
│   ├── 闭源：GPT-4V/4o（2023-2024）、Gemini（2023-2025）、Claude 3.5（2024）
│   └── 开源：LLaVA（2023-2024）、InternVL（2024-2025）、Qwen-VL（2024-2025）
│
├── 视频理解与生成
│   ├── 视频理解：TimeSformer → VideoMAE → InternVideo2
│   └── 视频生成：Sora（2024）→ 开源复现（Open-Sora、CogVideoX）
│
└── 语音多模态
    ├── 语音识别：Whisper（2022）→ Whisper v3（2023）
    ├── 语音生成：VALL-E → CosyVoice → Fish Speech
    └── 端到端语音 LLM：GPT-4o 原生语音、Qwen2-Audio
```
