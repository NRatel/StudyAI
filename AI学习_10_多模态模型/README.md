# 10 多模态模型

多模态模型解决的是：AI 不只读文字，还能看图、听音频、理解视频，并把这些信息放到同一个任务里处理。

{{img:ch10_00_multimodal_shared_space}}

## 先抓住这条线

```text
图像 / 语音 / 视频 → 编码成向量或 token → 接入语言模型 → 输出文字、动作或新内容
```

语言模型像一个强大的中枢，但它本来只懂 token。多模态模型要做的，就是把不同类型的信息翻译成它能处理的表示。
