# 论文与 FAQ

这个模块的论文主线是：生成模型如何从学习分布，到通过扩散去噪生成高质量内容。

## 推荐了解的经典工作

| 主题 | 可以关注什么 |
|------|--------------|
| VAE | 潜在空间如何用于生成 |
| GAN | 生成器和判别器如何对抗训练 |
| DDPM | 扩散模型如何通过去噪生成 |
| Latent Diffusion | 为什么在潜在空间生成更省成本 |
| DiT | Transformer 如何进入扩散模型 |

## FAQ

### 扩散模型是在拼接训练图片吗？

不是。它学习的是数据分布和去噪方向，不是简单把训练图片剪贴组合。

### 为什么同一个 Prompt 每次结果不同？

因为生成通常从随机噪声开始。随机种子不同，去噪路径也会不同。

### Prompt 写得越长越好吗？

不一定。清晰、具体、结构化通常比堆很多形容词更有效。

### 生图模型能证明某件事发生过吗？

不能。生成图像是合成内容，不能当作现实证据。
