# 06 生成模型与扩散模型

这个模块回答：AI 为什么能生成图片、声音和视频？核心不是“凭空想象”，而是模型学到了数据分布，然后从这个分布里采样。

## 先抓住这条线

```text
生成模型：学习数据长什么样
VAE / GAN / Flow：早期重要路线
扩散模型：先加噪，再学会一步步去噪
条件生成：用文字、图像、姿态等控制结果
```
