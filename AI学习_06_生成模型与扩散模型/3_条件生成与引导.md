# 条件生成与引导

> **前置知识**：[扩散模型原理](2_扩散模型原理.md)
>
> **本节目标**：理解文字、图片、姿态等条件如何控制生成结果。

---

## 1. 条件生成：不是随便生成，而是按要求生成

无条件生成只需要“生成一张像样的图”。条件生成则要满足额外要求：

```text
一只穿宇航服的猫，在月球上看地球
```

这里的文字就是条件。模型不仅要生成图片，还要让图片和条件匹配。

## 2. 文本条件：Prompt 怎么进入模型

在文本生成图像系统中，Prompt 通常会先经过文本编码器，变成向量表示。

扩散模型去噪时，会参考这些文本向量，决定往哪个方向生成。

直白地说：

> 文本条件像一张创作说明书，去噪过程每一步都会参考它。

## 3. Classifier-free Guidance：提示词拉得更用力

引导强度可以理解成模型听 Prompt 的用力程度。

- 引导太弱：图片更自由，但可能不听话。
- 引导太强：更贴 Prompt，但可能僵硬、过饱和或细节异常。

所以生图时调 guidance scale，本质是在平衡“符合提示”和“自然好看”。

## 4. 更多控制方式

除了文字，还可以用其他条件控制生成：

- 参考图：控制风格或主体。
- 边缘图：控制轮廓。
- 深度图：控制空间结构。
- 姿态骨架：控制人物动作。
- 分割图：控制区域布局。

这些方法的共同目标是：把“我想要什么”表达得更明确，让模型少猜一点。

## 5. Prompt 不是万能遥控器

Prompt 能影响生成，但不能保证模型完全按复杂要求执行。

尤其是数量、文字、精确空间关系、多主体交互，生成模型仍然容易出错。

所以复杂需求常需要：更明确的条件控制、多轮生成、局部重绘、后期编辑，甚至专门训练。
