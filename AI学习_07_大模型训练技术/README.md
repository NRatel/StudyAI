# AI学习：07 大模型训练技术

## 本模块导读

**学什么**：大模型训练的完整技术栈——分布式训练（数据并行、模型并行、3D并行）、混合精度与训练框架（DeepSpeed/FSDP）、数据工程与Tokenizer、对齐训练实践（SFT/RLHF/DPO）。

**为什么学**：理解了模型架构之后，下一个问题是"怎么训出来"。一个 175B 参数的模型，单卡放不下、单机算不完、数据要精心配比、训完还要对齐人类偏好。这些工程问题的解法，就是本模块的核心内容。

**学完能做什么**：
- 理解 DDP、模型并行、ZeRO、3D 并行的原理和适用场景
- 能配置 DeepSpeed / FSDP 进行分布式训练
- 理解混合精度训练的原理和实际应用
- 理解预训练数据工程和 Tokenizer（BPE/SentencePiece）的设计思路
- 理解 SFT → RLHF → DPO 的完整对齐训练流程
- 理解大模型训练配置背后的主要取舍：显存、吞吐、通信、稳定性和数据质量

**本模块默认前提**：多卡训练环境、显存是主要瓶颈、数据规模在 TB 级别。如果你只有单卡或数据量很小，本模块的部分内容（如 3D 并行、TB 级数据工程）可能暂时用不到，但理解这些概念对全局认知仍然重要。

**前置知识**：[05 GPT系列深度解析](../AI学习_05_GPT系列深度解析/README.md)

**预估学习时间**：8~10 小时

**重要程度**：★★★★

## 学完本模块，你应该能回答

1. 为什么大模型训练的核心瓶颈不只是“算力”，还包括显存、通信和数据？
2. 数据并行、张量并行、流水线并行、ZeRO 分别在切分什么？
3. FP16/BF16 混合精度为什么能加速训练，又为什么需要防止数值不稳定？
4. Tokenizer 和数据配比为什么会影响最终模型能力？
5. SFT、RLHF、DPO 从工程实践上分别需要什么数据和训练流程？

## 学习建议

1. **先读第1章**：分布式训练是后续所有内容的基础，搞清楚"为什么单卡不够"
2. **第2章配合实验**：混合精度和 DeepSpeed 最好在实际环境中跑一跑配置
3. **第3章理解思路**：数据工程偏实践，重点理解"为什么这么做"
4. **第4章串联全局**：对齐训练把前面模块的 RLHF/DPO 从原理拉到工程落地
