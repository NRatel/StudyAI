# 07 大模型训练技术

这个模块只抓住训练大模型的关键问题：模型太大、数据太多、计算太贵，所以必须把任务拆开、降低成本、管理数据，并让模型行为更可控。

## 先抓住这条线

```text
模型太大 → 分布式训练
计算太贵 → 混合精度
数据决定上限 → 数据工程和 Tokenizer
会续写还不够 → 对齐训练
```

## 本模块读法

| 顺序 | 章节 | 先看懂什么 |
|------|------|------------|
| 1 | [分布式训练](1_分布式训练.md) | 为什么单卡装不下、训不动 |

| 3 | [数据工程与 Tokenizer](3_数据工程与Tokenizer.md) | 数据和切词如何影响模型能力 |
| 4 | [对齐训练实践](4_对齐训练实践.md) | 训练流程如何从预训练走向助手 |
