# CNN 卷积神经网络

## 图像为什么需要卷积
### 核心问题：图像不能直接当一长串数字处理

一张 224 x 224 x 3 的彩色图片有 150,528 个像素值。如果直接用全连接网络处理，参数量会爆炸。

更糟的是，全连接网络把图像摊平成一维向量后，会丢掉一个关键信息：**空间邻近关系**。

在图像里，旁边的像素通常相关。猫耳朵、眼睛、胡须这些局部结构，不是随机散落的数字，而是空间模式。CNN 的核心就是利用这种结构。

### CNN 的三个核心思想

| 思想 | 直觉 | 解决什么问题 |
|------|------|--------------|
| 局部连接 | 先看小区域，不必一口吞下整张图 | 利用图像局部结构 |
| 权值共享 | 同一个探测器在整张图上滑动 | 大幅减少参数量 |
| 层级特征 | 低层看边缘，高层看部件，再高层看物体 | 从局部组合出语义 |

可以把卷积核想象成一个小小的"模式探测器"：它在图像上滑动，遇到像边缘、纹理、角点这样的模式就响应强，没遇到就响应弱。

## CNN 的核心机制
### 1. 卷积运算：滑动窗口做加权求和

严格来说，深度学习框架里的"卷积"多数是**互相关**：卷积核不翻转，直接与窗口内像素逐元素相乘再求和。

对二维输入 $X$ 和卷积核 $K$：

$$Y[i,j] = \sum_m\sum_n X[i+m,j+n]\cdot K[m,n]$$

直觉：

```
卷积核覆盖一小块区域
→ 每个位置相乘
→ 全部加起来
→ 得到输出特征图上的一个值
```

卷积核滑过整张图，就得到一张**特征图（Feature Map）**。特征图表示：某种局部模式在各个位置出现得有多强。

### 2. 感受野：一个输出能看到多大范围

**感受野（Receptive Field）**指某个神经元能"看到"输入图像中的多大区域。

- 单个 3 x 3 卷积只看局部 3 x 3 区域。
- 多层卷积叠起来后，高层神经元能间接看到更大区域。

这解释了 CNN 的层级特征：

```
低层：边缘、颜色、纹理
中层：眼睛、轮廓、局部部件
高层：猫、车、人脸等语义对象
```

### 3. Padding、Stride 和输出尺寸

CNN 常见三个参数：

| 参数 | 含义 | 直觉 |
|------|------|------|
| Kernel Size | 卷积核大小 | 探测器一次看多大 |
| Stride | 滑动步长 | 探测器每次跳多远 |
| Padding | 边缘补零 | 是否保留边缘信息和空间尺寸 |

输出尺寸公式：

$$H_{\text{out}} = \left\lfloor \frac{H_{\text{in}} + 2P - K}{S} \right\rfloor + 1$$

其中 $H_{\text{in}}$ 是输入高度，$K$ 是卷积核大小，$P$ 是 padding，$S$ 是 stride。宽度同理。

不用死记公式，抓住直觉即可：

- padding 越大，输出越大。
- stride 越大，输出越小。
- kernel 越大，边缘可滑动的位置越少，输出越小。

### 4. 多通道卷积：不是只看灰度图

彩色图像有 RGB 三个通道。卷积核也要有对应的输入通道。

如果输入是 $H \times W \times C_{\text{in}}$，一个卷积核的形状通常是：

$$K \times K \times C_{\text{in}}$$

它会同时看所有输入通道，输出一张特征图。

如果有 $C_{\text{out}}$ 个卷积核，就会输出 $C_{\text{out}}$ 张特征图。每个卷积核学习一种不同的模式探测器。

### 5. 池化层：压缩空间，保留强信号

池化（Pooling）常用于降低特征图尺寸。

**最大池化（Max Pooling）**取窗口内最大值：

```
[[1, 3],
 [2, 0]] → 3
```

直觉：只关心"这个模式有没有强烈出现"，不太关心精确位置。

池化的作用：

- 降低计算量
- 增加一定的位置鲁棒性
- 扩大后续层的感受野

现代 CNN 中，池化有时会被 stride 卷积替代，但"逐步降低空间分辨率、提高语义抽象度"这个思想仍然重要。

### 6. 经典 CNN 演进：记思想，不背结构

| 模型 | 关键思想 | 为什么重要 |
|------|----------|------------|
| LeNet-5 | 卷积 + 池化 + 全连接 | 奠定 CNN 基本范式 |
| AlexNet | ReLU、Dropout、大数据和 GPU | 引爆深度学习时代 |
| VGG | 多个 3 x 3 小卷积堆叠 | 证明深度和规整结构很有效 |
| Inception | 多尺度卷积分支 | 同时捕捉不同大小的模式 |
| ResNet | 残差连接 | 让很深的网络可以稳定训练 |

其中最值得记住的是 **ResNet 的残差连接**：

$$y = F(x) + x$$

它让网络可以学习"在输入基础上改一点"，而不是每层都从零学习完整变换。反向传播时，$+x$ 也给梯度提供了一条更直接的路径，缓解深层网络训练困难。

### 7. 1 x 1 卷积：通道混合器

1 x 1 卷积看起来像"没看邻居"，但它非常有用：它在每个空间位置上混合不同通道。

直觉：

```
空间位置不变
通道之间重新组合
```

常见用途：

- 降低通道数，减少计算量
- 提升通道数，增加表达能力
- 在 Inception、ResNet bottleneck 等结构中做通道变换

## 常见误区

**误区 1：CNN 只能用于图像。**

不是。只要数据有局部结构，卷积就可能有用。例如音频、时间序列、文本的局部 n-gram 模式都可以用卷积建模。

**误区 2：卷积核一开始就会识别猫眼睛。**

不会。低层卷积核通常学到边缘、颜色、纹理；更高层才逐渐组合成部件和语义对象。

**误区 3：池化只是为了省计算。**

省计算是一方面，更重要的是引入位置鲁棒性：模式稍微挪一点，模型仍能认出来。

## 本节小结

| 概念 | 一句话总结 |
|------|------------|
| 局部连接 | 每个神经元只看局部区域 |
| 权值共享 | 同一个卷积核在整张图上复用 |
| 特征图 | 某种模式在各位置的响应强度 |
| 感受野 | 一个输出能看到的输入范围 |
| 池化 | 压缩空间尺寸，保留强信号 |
| 残差连接 | 给深层网络提供更容易学习和传梯度的路径 |
| 1 x 1 卷积 | 在每个位置上混合通道 |
