# 论文与 FAQ

这个模块的论文可以当作“架构为什么这样设计”的历史线索，不必一开始深读所有公式。

## 推荐了解的经典工作

| 主题 | 可以关注什么 |
|------|--------------|
| LeNet / AlexNet | CNN 如何让图像识别进入深度学习时代 |
| Seq2Seq | 编码器-解码器如何处理变长输入输出 |
| LSTM / GRU | 门控如何改善 RNN 的长距离记忆 |
| Attention in Seq2Seq | 注意力如何缓解固定向量瓶颈 |

## FAQ

### CNN 现在是不是过时了？

不是。Transformer 在很多视觉任务中很强，但 CNN 的局部归纳偏置仍然有价值，尤其在数据规模、算力或特定视觉任务中。

### RNN 为什么不再是大语言模型主流？

主要是并行能力弱。RNN 必须按顺序读，Transformer 更容易大规模训练。

### LSTM 和 GRU 还需要学吗？

需要了解。它们帮助你理解序列建模、记忆、门控这些核心问题，也能解释 Transformer 为什么有吸引力。

### 编码器-解码器是不是只用于翻译？

不是。任何“输入一段东西，输出另一段东西”的任务都可以用这个框架理解，比如摘要、问答、语音识别。
