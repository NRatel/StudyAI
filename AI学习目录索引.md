# AI 及大模型原理 — 学习目录索引

这份索引的作用不是催你把所有章节一次性读完，而是帮你选路线。AI 原理很大，合理的读法是：先抓主干，再按兴趣和问题回补。读每个概念时优先问四件事：它是什么，为什么出现，解决什么问题，和前后技术有什么关系。

## 四条学习路径

### 路径 A：核心原理快速路径

目标：先抓住大语言模型到底怎么工作。

```
01 神经网络基础
  → 03 注意力机制与 Transformer
  → 04 预训练语言模型演进
  → 05 GPT 系列深度解析
```

适合：想先抓住 ChatGPT、Claude、DeepSeek 这类大语言模型主干的学习者。

提示：`02 经典网络架构` 是背景桥梁，不是硬前置。如果读 03 时对“编码器-解码器”“序列建模”没概念，再回补 02。

### 路径 B：完整模型原理路径

目标：把神经网络、语言模型、生成模型和多模态模型串成完整“模型谱系”。

```
01 神经网络基础
  → 02 经典网络架构
  → 03 注意力机制与 Transformer
  → 04 预训练语言模型演进
  → 05 GPT 系列深度解析
  → 06 生成模型与扩散模型

```

适合：希望把“AI 模型谱系”串起来的人，尤其是想同时看懂文本、图像、视频生成路线的人。

### 路径 C：完整大模型系统路径

目标：理解大模型从训练、推理、接入外部知识到工具行动的完整系统链路。


适合：主要关心大模型怎么训练、怎么部署、怎么接知识库、怎么做推理和 Agent 的学习者。

数学和代码都不建议抢跑：

- [术语速查](附录_术语速查.md)：看到陌生概念时先查一句话定义和常见误解。
- [数学基础速览](附录_数学基础速览.md)：遇到向量、梯度、概率、熵这些概念时再查。
- [代码示例集](附录_代码示例集.md)：概念读通后，用来验证直觉，不作为主线学习材料。
- [整体知识体系思维导图](整体知识体系思维导图.md)：读完几个模块后回看，帮助把知识点挂回全局。

## 怎么判断自己该继续还是回补

读每个模块时，只要能用自己的话解释模块 README 里的关键问题，就可以继续往后走。不必追求把每个公式、每个工程细节都一次性吃透。

如果连续两节都出现“术语认识但关系不清”的感觉，先回到最近的前置模块补地基。AI 学习最怕的不是慢，而是带着一串半懂的词继续往前冲。
