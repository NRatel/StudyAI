# 08 大模型推理与部署

这个模块回答：模型训练好以后，为什么生成仍然慢、贵、占显存？推理优化的目标就是在尽量不损失质量的情况下，让模型更快、更省。

## 先抓住这条线

```text
逐 token 生成 → 重复计算多 → KV Cache 复用历史
模型太大 → 量化减少体积
请求很多 → 批处理和推理框架提高吞吐
```
