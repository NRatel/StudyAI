# 01 神经网络基础

这个模块回答一个最根本的问题：模型到底是怎么“学会”的？

先不用把神经网络想成大脑。更直白地看，它是一套会调参数的函数：输入进来，模型给出答案；答案错了，损失函数打分；反向传播把错误传回去；优化器再把参数往更好的方向挪一点。

{{img:ch01_00_nn_learning_loop}}

## 先抓住这条线

```text
输入 → 加权计算 → 输出 → 计算损失 → 反向传播 → 更新参数
```

如果这条线通了，后面看 Transformer、GPT、扩散模型都会轻松很多。
