# AI学习：01 神经网络基础

> 读本模块时，先抓住神经网络如何从“输入、预测、损失、梯度、参数更新”形成一个闭环。公式用于说明这个闭环，不需要第一次就把完整推导吃透。

## 本模块导读

**学什么**：神经网络是几乎所有现代 AI 模型的基础构件。本模块从最简单的单个神经元（感知机）出发，逐步构建到多层网络，了解神经网络"学习"的完整机制。

**为什么学**：后续所有模块（CNN、RNN、Transformer、GPT、扩散模型……）都是在本模块的基础上搭建的。如果把 AI 比作一栋大楼，本模块就是地基和承重结构。

{{img:ch01_00_nn_learning_loop}}

**读完能看懂的主线**：
- 理解神经网络如何从数据中"学习"：前向传播做预测，损失函数衡量好坏，反向传播算梯度，优化器更新参数
- 能用概念和公式解释一个简单多层网络的计算流程
- 看懂后续 CNN、RNN、Transformer 中反复出现的基础部件
- 为后续理解 CNN、RNN、Transformer 打下基础

**前置知识**：基本函数、向量和求导直觉。数学知识在文中穿插讲解，详细推导见 [附录：数学基础速览](../附录_数学基础速览.md)。如果数学基础薄弱，优先查阅附录 1.1（向量与矩阵运算）、3.3（链式法则）、3.5（梯度下降），建立最低限度的数学直觉后再开始正文阅读。

**预估阅读时间**：8~10 小时

## 读完能解释

1. 为什么单个感知机只能解决线性可分问题，而多层网络能表达更复杂的关系？
2. 如果去掉激活函数，多层神经网络为什么会退化成一个线性模型？
3. 损失函数、反向传播、梯度下降和优化器如何组成一个完整的“学习闭环”？
4. 过拟合为什么会发生？Dropout、权重衰减、早停分别在抑制什么？
