# 激活函数

> **前置知识**：[感知机与多层网络](1_感知机与多层网络.md)
>
> **本节目标**：理解为什么神经网络必须加入非线性，以及 Sigmoid、Tanh、ReLU、GELU、Softmax 分别解决什么问题。

---

## 1. 风趣易懂的直觉讲解

### 1.1 为什么需要激活函数

神经网络的每一层通常先做一次线性变换：

$$z = Wx + b$$

如果层与层之间没有激活函数，那么多层线性变换可以合并成一层：

$$W_2(W_1x + b_1) + b_2 = W'x + b'$$

也就是说：

```text
没有激活函数：
100 层线性网络 ≈ 1 层线性网络
```

这就很尴尬了。模型看起来很深，表达能力却没有真正变复杂。

**激活函数的作用**：在线性变换之后加入非线性，让网络能够学习曲线、边界、组合关系等复杂模式。

### 1.2 一个阀门类比

可以把激活函数想成信号阀门：

- Sigmoid：把信号压到 0 到 1，像概率开关。
- Tanh：把信号压到 -1 到 1，像带正负方向的开关。
- ReLU：正数通过，负数截断，像单向阀。
- GELU/SiLU：不是硬截断，而是平滑地决定通过多少。

线性层负责“混合信息”，激活函数负责“改变形状”。两者交替出现，网络才有深度学习的表达能力。

## 2. 准确概念定义与核心原理

### 2.1 Sigmoid

$$\sigma(x)=\frac{1}{1+e^{-x}}$$

输出范围是 $(0,1)$，因此很适合表示概率。

| 特点 | 说明 |
|------|------|
| 优点 | 输出像概率，适合二分类输出层 |
| 缺点 | 输入很大或很小时曲线变平，容易梯度消失 |
| 常见位置 | 二分类输出层、门控结构 |

注意：Sigmoid 现在很少用于深层网络的隐藏层，因为它两端饱和，深层反向传播时梯度容易越来越小。

### 2.2 Tanh

$$\tanh(x)=\frac{e^x-e^{-x}}{e^x+e^{-x}}$$

输出范围是 $(-1,1)$。它可以看成“零中心版 Sigmoid”。

| 特点 | 说明 |
|------|------|
| 优点 | 输出有正有负，比 Sigmoid 更适合作为中间状态 |
| 缺点 | 输入绝对值很大时仍会饱和，仍可能梯度消失 |
| 常见位置 | RNN/LSTM 的隐藏状态 |

### 2.3 ReLU

$$\text{ReLU}(x)=\max(0,x)$$

ReLU 的规则极其简单：正数原样通过，负数变成 0。

| 特点 | 说明 |
|------|------|
| 优点 | 简单、快，正半轴梯度稳定 |
| 缺点 | 负半轴梯度为 0，可能出现“死神经元” |
| 常见位置 | CNN、MLP 等一般隐藏层 |

**死神经元**：如果某个神经元长期只收到负输入，它的输出一直是 0，梯度也一直是 0，就很难再被训练回来。

### 2.4 Leaky ReLU / PReLU

Leaky ReLU 是 ReLU 的小修补：负半轴不直接变 0，而是保留一个很小的斜率。

$$\text{LeakyReLU}(x)=
\begin{cases}
x, & x>0\\
\alpha x, & x\le 0
\end{cases}
$$

它的目的很明确：**让负半轴也有一点梯度，降低死神经元风险**。

PReLU 则进一步让 $\alpha$ 变成可学习参数。初学阶段只需知道：它们都是 ReLU 的改良版。

### 2.5 GELU 与 SiLU

GELU：

$$\text{GELU}(x)=x\cdot \Phi(x)$$

SiLU / Swish：

$$\text{SiLU}(x)=x\cdot \sigma(x)$$

它们都像“平滑版 ReLU”：正数大多通过，负数大多被抑制，但不是硬切一刀。

| 函数 | 直觉 | 常见位置 |
|------|------|----------|
| GELU | 按输入大小平滑控制通过比例 | BERT、GPT 等 Transformer |
| SiLU | 用 Sigmoid 做平滑门控 | LLaMA 等现代模型 |

学习本章不需要掌握 GELU 的展开计算。记住它的角色即可：**Transformer 里常用的平滑非线性激活函数**。

### 2.6 Softmax

Softmax 把一组分数转换成概率分布：

$$\text{softmax}(z_i)=\frac{e^{z_i}}{\sum_j e^{z_j}}$$

它有两个关键性质：

- 每个输出都大于 0。
- 所有输出加起来等于 1。

因此 Softmax 常用于：

- 多分类输出层。
- 注意力机制中，把相关性分数转成注意力权重。

Softmax 和前面的 ReLU/GELU 不太一样：它不是对每个元素单独处理，而是对整个向量一起归一化。

## 3. 如何选择激活函数

| 场景 | 常见选择 | 理由 |
|------|----------|------|
| 一般隐藏层 | ReLU | 简单、稳定、效果好 |
| Transformer 隐藏层 | GELU 或 SiLU | 平滑，适合现代大模型结构 |
| RNN/LSTM 状态 | Tanh / Sigmoid | 状态值和门控需要有界范围 |
| 二分类输出层 | Sigmoid | 输出单个正类概率 |
| 多分类输出层 | Softmax | 输出类别概率分布 |
| 回归输出层 | 通常不加激活 | 保留任意实数输出 |

一句话判断：

```text
隐藏层：ReLU / GELU
输出层：看任务需要什么形式的答案
```

## 本节小结

| 概念 | 一句话总结 |
|------|-----------|
| **激活函数** | 给网络加入非线性，让多层网络真正有意义 |
| **Sigmoid** | 输出 0 到 1，适合二分类概率 |
| **Tanh** | 输出 -1 到 1，常用于循环网络状态 |
| **ReLU** | 正数通过、负数截断，简单高效 |
| **Leaky ReLU** | 给负半轴一点梯度，减少死神经元 |
| **GELU / SiLU** | 平滑门控，常见于 Transformer |
| **Softmax** | 把一组分数变成概率分布 |
