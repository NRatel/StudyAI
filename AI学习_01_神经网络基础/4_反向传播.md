# 反向传播

## 直觉与概述

### 核心问题

你训练了一个神经网络，前向传播算出了预测值，损失函数告诉你"预测有多差"。现在问题来了：**网络有成千上万个参数（权重和偏置），每一个该往哪个方向调？调多少？**

如果只有一个参数，你可以试试加一点、减一点，看看损失是变好还是变坏。但如果有一百万个参数，每个都要"试一试"，计算成本根本无法接受。

反向传播（Backpropagation）解决的就是这个问题：**一次前向传播 + 一次反向传播，就能同时算出所有参数的梯度**。

### 生产线类比

想象一条三道工序的生产线：

```
原材料 → [工序A: 切割] → [工序B: 打磨] → [工序C: 组装] → 成品（不合格！）
```

成品出了问题，质检员要"反向排查"每道工序对最终缺陷的贡献度：

1. 先看工序 C：组装环节自身引入了多少问题？
2. 再看工序 B：打磨环节的偏差，经过工序 C 放大后，对成品造成了多少影响？
3. 最后看工序 A：切割环节的偏差，经过工序 B 和 C 逐级放大后，对成品造成了多少影响？

这就是反向传播的思路——**从输出端（损失）开始，逐层往回追溯，算出每一层、每一个参数对最终损失的"贡献度"（梯度）**。

### 一句话总结

> **反向传播 = 链式法则的系统化应用。** 它利用计算图的结构，从损失出发，沿着计算路径反向传递梯度，一次性算出所有参数的偏导数。

## 严谨定义与原理

### 计算图：把前向传播画成图

神经网络的前向传播可以拆成一系列基本运算（矩阵乘法、加法、激活函数等），每个运算是图中的一个节点，数据沿着有向边流动。这个有向无环图（DAG）就叫**计算图**。

以一个简单的单神经元为例：$y = \sigma(wx + b)$

```
     x ──→ [乘法: z1 = w*x] ──→ [加法: z2 = z1 + b] ──→ [Sigmoid: y = σ(z2)] ──→ [损失: L = (y - t)²]
     w ──↗                        b ──↗                                               t ──↗
```

每个节点知道两件事：
1. **前向**：给定输入，怎么算输出
2. **反向**：给定"输出的梯度"（从后面传回来的），怎么算"输入的梯度"（传给前面的）

这就是反向传播的全部原理。没有魔法，只是把链式法则在计算图上系统化地执行。

### 链式法则复习

> 详细推导见 [附录：数学基础速览 - 3.3 链式法则](../附录_数学基础速览.md)

**单变量链式法则**：如果 $y = f(g(x))$，令 $u = g(x)$，则：

$$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} = f'(u) \cdot g'(x)$$

**多变量链式法则**：如果 $L$ 通过多条路径依赖于 $w$，梯度是各路径的**加和**：

$$\frac{\partial L}{\partial w} = \sum_{i} \frac{\partial L}{\partial z_i} \cdot \frac{\partial z_i}{\partial w}$$

其中 $z_i$ 是所有直接依赖 $w$ 的中间变量。

**核心直觉**：链式法则说的是——要算最终输出对某个输入的敏感度，就把沿途每一步的"放大倍数"（局部导数）乘起来。

### 完整手算示例：2 层网络

我们用一个具体的 2 层网络，带入数值，完整走一遍前向传播和反向传播。**不跳任何步骤。**

#### 网络结构

```
输入层（2个）→ 隐藏层（2个神经元，Sigmoid 激活）→ 输出层（1个神经元，Sigmoid 激活）→ MSE 损失
```

#### 初始参数

```
输入:      x₁ = 0.05,  x₂ = 0.10
目标输出:   t = 0.01

第一层权重:
  w₁ = 0.15,  w₂ = 0.20    （连接 x₁,x₂ 到隐藏神经元 h₁）
  w₃ = 0.25,  w₄ = 0.30    （连接 x₁,x₂ 到隐藏神经元 h₂）
第一层偏置:
  b₁ = 0.35                 （隐藏神经元 h₁ 的偏置）
  b₂ = 0.35                 （隐藏神经元 h₂ 的偏置）

第二层权重:
  w₅ = 0.40,  w₆ = 0.45    （连接 h₁,h₂ 到输出 o）
第二层偏置:
  b₃ = 0.60                 （输出神经元 o 的偏置）
```

用矩阵形式表示：

$$W_1 = \begin{bmatrix} w_1 & w_2 \\ w_3 & w_4 \end{bmatrix} = \begin{bmatrix} 0.15 & 0.20 \\ 0.25 & 0.30 \end{bmatrix}, \quad \mathbf{b}_1 = \begin{bmatrix} 0.35 \\ 0.35 \end{bmatrix}$$

$$W_2 = \begin{bmatrix} w_5 & w_6 \end{bmatrix} = \begin{bmatrix} 0.40 & 0.45 \end{bmatrix}, \quad b_3 = 0.60$$

#### 第一部分：前向传播

**步骤 1：隐藏层的线性变换**

$$z_{h1} = w_1 \cdot x_1 + w_2 \cdot x_2 + b_1 = 0.15 \times 0.05 + 0.20 \times 0.10 + 0.35$$
$$= 0.0075 + 0.0200 + 0.35 = 0.3775$$

$$z_{h2} = w_3 \cdot x_1 + w_4 \cdot x_2 + b_2 = 0.25 \times 0.05 + 0.30 \times 0.10 + 0.35$$
$$= 0.0125 + 0.0300 + 0.35 = 0.3925$$

**步骤 2：隐藏层的 Sigmoid 激活**

$$h_1 = \sigma(z_{h1}) = \frac{1}{1 + e^{-0.3775}} = \frac{1}{1 + 0.6856} = \frac{1}{1.6856} = 0.5933$$

$$h_2 = \sigma(z_{h2}) = \frac{1}{1 + e^{-0.3925}} = \frac{1}{1 + 0.6753} = \frac{1}{1.6753} = 0.5969$$

**步骤 3：输出层的线性变换**

$$z_o = w_5 \cdot h_1 + w_6 \cdot h_2 + b_3 = 0.40 \times 0.5933 + 0.45 \times 0.5969 + 0.60$$
$$= 0.2373 + 0.2686 + 0.60 = 1.1059$$

**步骤 4：输出层的 Sigmoid 激活**

$$o = \sigma(z_o) = \frac{1}{1 + e^{-1.1059}} = \frac{1}{1 + 0.3311} = \frac{1}{1.3311} = 0.7514$$

**步骤 5：计算损失（MSE）**

$$L = \frac{1}{2}(t - o)^2 = \frac{1}{2}(0.01 - 0.7514)^2 = \frac{1}{2}(−0.7414)^2 = \frac{1}{2} \times 0.5497 = 0.2748$$

> 注：这里用 $\frac{1}{2}(t-o)^2$ 而不是 $(t-o)^2$，是因为求导时 $\frac{1}{2}$ 与平方的 2 相消，公式更简洁。

**前向传播汇总**：

| 变量 | 值 |
|------|-----|
| $z_{h1}$ | 0.3775 |
| $z_{h2}$ | 0.3925 |
| $h_1$ | 0.5933 |
| $h_2$ | 0.5969 |
| $z_o$ | 1.1059 |
| $o$ | 0.7514 |
| $L$ | 0.2748 |

#### 第二部分：反向传播

现在从损失 $L$ 出发，逐步反向计算每个参数的梯度。

我们需要用到的 Sigmoid 导数公式：

$$\sigma'(z) = \sigma(z) \cdot (1 - \sigma(z))$$

**步骤 1：损失对输出的梯度**

$$\frac{\partial L}{\partial o} = \frac{\partial}{\partial o}\left[\frac{1}{2}(t - o)^2\right] = -(t - o) = -(0.01 - 0.7514) = 0.7414$$

**步骤 2：输出对 $z_o$ 的梯度（Sigmoid 导数）**

$$\frac{\partial o}{\partial z_o} = o \cdot (1 - o) = 0.7514 \times (1 - 0.7514) = 0.7514 \times 0.2486 = 0.1868$$

**步骤 3：损失对 $z_o$ 的梯度（链式法则）**

$$\delta_o = \frac{\partial L}{\partial z_o} = \frac{\partial L}{\partial o} \cdot \frac{\partial o}{\partial z_o} = 0.7414 \times 0.1868 = 0.1385$$

> $\delta_o$ 称为输出层的**误差项**（error term），它是从损失传回到该层线性输出的梯度。后续计算全部基于它。

**步骤 4：第二层权重的梯度**

因为 $z_o = w_5 \cdot h_1 + w_6 \cdot h_2 + b_3$，所以：

$$\frac{\partial z_o}{\partial w_5} = h_1 = 0.5933, \quad \frac{\partial z_o}{\partial w_6} = h_2 = 0.5969, \quad \frac{\partial z_o}{\partial b_3} = 1$$

应用链式法则：

$$\frac{\partial L}{\partial w_5} = \delta_o \cdot h_1 = 0.1385 \times 0.5933 = 0.0821$$

$$\frac{\partial L}{\partial w_6} = \delta_o \cdot h_2 = 0.1385 \times 0.5969 = 0.0827$$

$$\frac{\partial L}{\partial b_3} = \delta_o \cdot 1 = 0.1385$$

**步骤 5：梯度回传到隐藏层输出**

梯度需要继续向前传播。$z_o$ 对 $h_1$ 和 $h_2$ 的偏导：

$$\frac{\partial z_o}{\partial h_1} = w_5 = 0.40, \quad \frac{\partial z_o}{\partial h_2} = w_6 = 0.45$$

$$\frac{\partial L}{\partial h_1} = \delta_o \cdot w_5 = 0.1385 \times 0.40 = 0.0554$$

$$\frac{\partial L}{\partial h_2} = \delta_o \cdot w_6 = 0.1385 \times 0.45 = 0.0623$$

**步骤 6：隐藏层的误差项（穿过 Sigmoid）**

$$\frac{\partial h_1}{\partial z_{h1}} = h_1 \cdot (1 - h_1) = 0.5933 \times 0.4067 = 0.2413$$

$$\frac{\partial h_2}{\partial z_{h2}} = h_2 \cdot (1 - h_2) = 0.5969 \times 0.4031 = 0.2406$$

$$\delta_{h1} = \frac{\partial L}{\partial z_{h1}} = \frac{\partial L}{\partial h_1} \cdot \frac{\partial h_1}{\partial z_{h1}} = 0.0554 \times 0.2413 = 0.0134$$

$$\delta_{h2} = \frac{\partial L}{\partial z_{h2}} = \frac{\partial L}{\partial h_2} \cdot \frac{\partial h_2}{\partial z_{h2}} = 0.0623 \times 0.2406 = 0.0150$$

**步骤 7：第一层权重的梯度**

因为 $z_{h1} = w_1 \cdot x_1 + w_2 \cdot x_2 + b_1$，所以：

$$\frac{\partial L}{\partial w_1} = \delta_{h1} \cdot x_1 = 0.0134 \times 0.05 = 0.0007$$

$$\frac{\partial L}{\partial w_2} = \delta_{h1} \cdot x_2 = 0.0134 \times 0.10 = 0.0013$$

$$\frac{\partial L}{\partial b_1} = \delta_{h1} = 0.0134$$

同理，因为 $z_{h2} = w_3 \cdot x_1 + w_4 \cdot x_2 + b_2$：

$$\frac{\partial L}{\partial w_3} = \delta_{h2} \cdot x_1 = 0.0150 \times 0.05 = 0.0008$$

$$\frac{\partial L}{\partial w_4} = \delta_{h2} \cdot x_2 = 0.0150 \times 0.10 = 0.0015$$

$$\frac{\partial L}{\partial b_2} = \delta_{h2} = 0.0150$$

#### 反向传播梯度汇总

| 参数 | 梯度 $\partial L / \partial (\cdot)$ |
|------|--------------------------------------|
| $w_5$ | 0.0821 |
| $w_6$ | 0.0827 |
| $b_3$ | 0.1385 |
| $w_1$ | 0.0007 |
| $w_2$ | 0.0013 |
| $b_1$ | 0.0134 |
| $w_3$ | 0.0008 |
| $w_4$ | 0.0015 |
| $b_2$ | 0.0150 |

**观察**：第一层的梯度（$w_1$ 到 $w_4$）比第二层的梯度（$w_5$, $w_6$）小了两个数量级。这不是巧合——这正是**梯度消失**的早期信号，后面会详细讨论。

#### 参数更新（一步梯度下降）

取学习率 $\eta = 0.5$，更新规则 $\theta_{\text{new}} = \theta_{\text{old}} - \eta \cdot \frac{\partial L}{\partial \theta}$：

$$w_5^{\text{new}} = 0.40 - 0.5 \times 0.0821 = 0.3590$$

$$w_6^{\text{new}} = 0.45 - 0.5 \times 0.0827 = 0.4087$$

$$w_1^{\text{new}} = 0.15 - 0.5 \times 0.0007 = 0.1497$$

以此类推。每次迭代（前向 + 反向 + 更新）后，损失都会减小一点。重复上万次，网络就"学会了"。

### 反向传播的一般算法

把上面的手算过程推广到任意 $N$ 层网络。

**符号约定**：
- 第 $l$ 层的权重矩阵 $W^{(l)}$，偏置 $\mathbf{b}^{(l)}$
- 第 $l$ 层的线性输出 $\mathbf{z}^{(l)} = W^{(l)} \mathbf{a}^{(l-1)} + \mathbf{b}^{(l)}$
- 第 $l$ 层的激活输出 $\mathbf{a}^{(l)} = f(\mathbf{z}^{(l)})$，其中 $f$ 是激活函数
- $\mathbf{a}^{(0)} = \mathbf{x}$（输入）

**前向传播**（从第 1 层到第 $N$ 层）：

$$\text{for } l = 1, 2, \dots, N: \quad \mathbf{z}^{(l)} = W^{(l)} \mathbf{a}^{(l-1)} + \mathbf{b}^{(l)}, \quad \mathbf{a}^{(l)} = f(\mathbf{z}^{(l)})$$

计算损失 $L = \mathcal{L}(\mathbf{a}^{(N)}, \mathbf{y})$

**反向传播**（从第 $N$ 层到第 1 层）：

1. **输出层误差项**：

$$\boldsymbol{\delta}^{(N)} = \frac{\partial L}{\partial \mathbf{z}^{(N)}} = \frac{\partial L}{\partial \mathbf{a}^{(N)}} \odot f'(\mathbf{z}^{(N)})$$

其中 $\odot$ 表示逐元素乘法（Hadamard 积）。

2. **隐藏层误差项**（梯度从后往前"流动"）：

$$\boldsymbol{\delta}^{(l)} = \left({W^{(l+1)}}^T \boldsymbol{\delta}^{(l+1)}\right) \odot f'(\mathbf{z}^{(l)}), \quad l = N-1, N-2, \dots, 1$$

这里 ${W^{(l+1)}}^T \boldsymbol{\delta}^{(l+1)}$ 把后一层的误差通过权重矩阵"回传"到当前层，$f'(\mathbf{z}^{(l)})$ 再乘上激活函数的导数。

3. **参数梯度**：

$$\frac{\partial L}{\partial W^{(l)}} = \boldsymbol{\delta}^{(l)} {\mathbf{a}^{(l-1)}}^T$$

$$\frac{\partial L}{\partial \mathbf{b}^{(l)}} = \boldsymbol{\delta}^{(l)}$$

**梯度流动的可视化**：

```
前向传播（从左到右）:
x ──→ [W₁,b₁] ──→ z⁽¹⁾ ──→ f(·) ──→ a⁽¹⁾ ──→ [W₂,b₂] ──→ z⁽²⁾ ──→ f(·) ──→ a⁽²⁾ ──→ L

反向传播（从右到左）:
     ∂L/∂W₁    δ⁽¹⁾    ×f'(z⁽¹⁾)      W₂ᵀδ⁽²⁾    δ⁽²⁾    ×f'(z⁽²⁾)      ∂L/∂a⁽²⁾
  ←───────── ←────── ←────────── ←────────── ←────── ←────────── ←──────────  ∂L/∂L=1
```

### 梯度消失与梯度爆炸

#### 问题的根源：连乘效应

从反向传播的公式可以看到，第 $l$ 层的误差项需要乘以后续所有层的激活函数导数和权重矩阵：

$$\boldsymbol{\delta}^{(l)} = \left(\prod_{k=l+1}^{N} {W^{(k)}}^T \cdot \text{diag}(f'(\mathbf{z}^{(k-1)}))\right) \cdot \frac{\partial L}{\partial \mathbf{a}^{(N)}} \odot f'(\mathbf{z}^{(l)})$$

这意味着从第 $N$ 层到第 $l$ 层，梯度经历了 $N - l$ 次矩阵乘法和 $N - l$ 次激活函数导数的缩放。

如果每次乘以的因子 **都小于 1**，连乘的结果会指数级趋近于 0——这就是**梯度消失**（Vanishing Gradients）。

如果每次乘以的因子 **都大于 1**，连乘的结果会指数级增长——这就是**梯度爆炸**（Exploding Gradients）。

#### 与 Sigmoid 激活函数的关系

Sigmoid 的导数：

$$\sigma'(z) = \sigma(z)(1 - \sigma(z))$$

这个导数的最大值出现在 $z = 0$ 处：$\sigma'(0) = 0.25$。也就是说，**每经过一层 Sigmoid，梯度最多保留原来的 25%**。

10 层 Sigmoid 网络中，从输出传到第 1 层的梯度上界：

$$0.25^{10} = 0.000000954 \approx 10^{-6}$$

梯度缩小了 100 万倍！这就是为什么早期深度网络（用 Sigmoid）几乎无法训练——前面几层的参数根本收不到有效的梯度信号。

#### ReLU 的解决方案

ReLU 的导数：

$$\text{ReLU}'(z) = \begin{cases} 1, & z > 0 \\ 0, & z \leq 0 \end{cases}$$

当 $z > 0$ 时，导数恒为 1，梯度可以无损传递。这就是 ReLU 成为现代网络默认激活函数的核心原因——它极大地缓解了梯度消失问题。

但 ReLU 也有自己的问题（"死亡 ReLU"：某些神经元输出永远为 0，梯度也永远为 0，再也无法更新），因此衍生出了 Leaky ReLU、GELU 等变体。

## Python 代码示例

### 示例 1：用 numpy 手动实现完整的前向+反向传播

这段代码严格对应上面的手算示例，你可以对照每一步的数值。

```python
import numpy as np

# ============================================================
# 手动实现 2 层网络的前向传播 + 反向传播（与手算示例完全对应）
# 网络结构：2 输入 → 2 隐藏(Sigmoid) → 1 输出(Sigmoid) → MSE 损失
# ============================================================

# --- Sigmoid 及其导数 ---
def sigmoid(z):
    return 1.0 / (1.0 + np.exp(-z))

def sigmoid_derivative(z):
    """Sigmoid 导数：σ'(z) = σ(z)(1 - σ(z))"""
    s = sigmoid(z)
    return s * (1.0 - s)

# ============================================================
# 初始参数（与手算完全一致）
# ============================================================
x = np.array([0.05, 0.10])   # 输入
t = np.array([0.01])          # 目标

# 第一层：2 → 2
W1 = np.array([[0.15, 0.20],   # w1, w2（连接到 h1）
               [0.25, 0.30]])  # w3, w4（连接到 h2）
b1 = np.array([0.35, 0.35])

# 第二层：2 → 1
W2 = np.array([[0.40, 0.45]])  # w5, w6
b2 = np.array([0.60])

learning_rate = 0.5

# ============================================================
# 前向传播
# ============================================================
# 第一层
z1 = W1 @ x + b1              # 线性变换：z = Wx + b
a1 = sigmoid(z1)              # 激活

# 第二层
z2 = W2 @ a1 + b2             # 线性变换
a2 = sigmoid(z2)              # 激活（即网络输出 o）

# 损失
loss = 0.5 * (t - a2) ** 2    # MSE 损失

print("=" * 60)
print("前向传播")
print("=" * 60)
print(f"  z1 (隐藏层线性输出)  = {z1}")
print(f"  a1 (隐藏层激活输出)  = {a1}")
print(f"  z2 (输出层线性输出)  = {z2}")
print(f"  a2 (输出层激活输出)  = {a2}")
print(f"  loss                 = {loss}")

# ============================================================
# 反向传播
# ============================================================
print("\n" + "=" * 60)
print("反向传播")
print("=" * 60)

# --- 输出层 ---
# 步骤 1: ∂L/∂a2 = -(t - a2)
dL_da2 = -(t - a2)
print(f"  ∂L/∂a2 = {dL_da2}")

# 步骤 2: ∂a2/∂z2 = sigmoid'(z2)
da2_dz2 = sigmoid_derivative(z2)
print(f"  ∂a2/∂z2 (sigmoid') = {da2_dz2}")

# 步骤 3: δ_o = ∂L/∂z2 = ∂L/∂a2 * ∂a2/∂z2
delta2 = dL_da2 * da2_dz2
print(f"  δ_o = ∂L/∂z2 = {delta2}")

# 步骤 4: 第二层参数梯度
# ∂L/∂W2 = δ_o · a1^T，∂L/∂b2 = δ_o
dL_dW2 = delta2[:, None] @ a1[None, :]   # (1,1) @ (1,2) = (1,2)
dL_db2 = delta2.copy()
print(f"  ∂L/∂W2 = {dL_dW2}  (对应 w5, w6 的梯度)")
print(f"  ∂L/∂b2 = {dL_db2}")

# --- 隐藏层 ---
# 步骤 5: 梯度回传到 a1
# ∂L/∂a1 = W2^T @ δ_o
dL_da1 = W2.T @ delta2       # (2,1) @ (1,) = (2,)
print(f"\n  ∂L/∂a1 (梯度回传) = {dL_da1}")

# 步骤 6: 穿过 Sigmoid
# δ_h = ∂L/∂z1 = ∂L/∂a1 * sigmoid'(z1)
da1_dz1 = sigmoid_derivative(z1)
print(f"  sigmoid'(z1)        = {da1_dz1}")
delta1 = dL_da1 * da1_dz1
```

### 示例 3：封装成可复用的 numpy 网络类

```python
import numpy as np

# ============================================================
# 通用的 numpy 手写神经网络（支持任意层数）
# 完整实现前向传播、反向传播、参数更新
# ============================================================

class NeuralNetwork:
    """
    一个用 numpy 从零实现的全连接神经网络。
    支持任意层数，使用 Sigmoid 激活和 MSE 损失。

    用法:
        nn = NeuralNetwork([2, 4, 4, 1])  # 2输入, 两个4节点隐藏层, 1输出
        nn.train(X, y, epochs=1000, lr=0.1)
    """

    def __init__(self, layer_sizes, seed=42):
        """
        参数:
            layer_sizes: 列表，如 [2, 3, 1] 表示 2输入→3隐藏→1输出
        """
        np.random.seed(seed)
        self.n_layers = len(layer_sizes) - 1  # 权重矩阵的数量
        self.weights = []
        self.biases = []

        # Xavier 初始化：让每层的输出方差保持一致，缓解梯度消失
        for i in range(self.n_layers):
            fan_in = layer_sizes[i]
            fan_out = layer_sizes[i + 1]
            scale = np.sqrt(2.0 / (fan_in + fan_out))
            W = np.random.randn(fan_out, fan_in) * scale
            b = np.zeros(fan_out)
            self.weights.append(W)
            self.biases.append(b)

    @staticmethod
    def sigmoid(z):
        # 数值稳定的 Sigmoid：避免 exp 溢出
        return np.where(z >= 0,
                        1.0 / (1.0 + np.exp(-z)),
                        np.exp(z) / (1.0 + np.exp(z)))

    @staticmethod
    def sigmoid_deriv(z):
        s = NeuralNetwork.sigmoid(z)
        return s * (1.0 - s)

    def forward(self, x):
        """
        前向传播。
        保存每层的 z（线性输出）和 a（激活输出），供反向传播使用。
        """
        self.z_list = []  # 每层的线性输出
        self.a_list = [x]  # 每层的激活输出，a_list[0] = 输入

        a = x
        for i in range(self.n_layers):
            z = self.weights[i] @ a + self.biases[i]
            a = self.sigmoid(z)
            self.z_list.append(z)
            self.a_list.append(a)

        return a  # 最后一层的输出

    def backward(self, y_true):
        """
        反向传播。
        返回每层权重和偏置的梯度。
        """
        output = self.a_list[-1]

        # 梯度列表
        dW_list = []
        db_list = []

        # --- 输出层 ---
        # MSE 损失: L = 0.5 * (y_true - output)^2
        # ∂L/∂output = -(y_true - output) = output - y_true
        dL_da = output - y_true

        # 从最后一层往前
        for i in reversed(range(self.n_layers)):
            z = self.z_list[i]
            a_prev = self.a_list[i]  # 前一层的激活输出

            # δ = ∂L/∂z = ∂L/∂a * sigmoid'(z)
            delta = dL_da * self.sigmoid_deriv(z)

            # ∂L/∂W = δ · a_prev^T
            dW = np.outer(delta, a_prev)
            db = delta.copy()

            # 插入到列表前面（因为我们是反向遍历）
            dW_list.insert(0, dW)
            db_list.insert(0, db)

            # 梯度继续回传：∂L/∂a_prev = W^T @ δ
            if i > 0:  # 不需要传到输入层
                dL_da = self.weights[i].T @ delta

        return dW_list, db_list

    def update(self, dW_list, db_list, lr):
        """用梯度下降更新参数。"""
        for i in range(self.n_layers):
            self.weights[i] -= lr * dW_list[i]
            self.biases[i] -= lr * db_list[i]

    def train(self, X, y, epochs=1000, lr=0.1, print_every=200):
        """
        训练网络。
        X: (n_samples, n_features) 的输入
        y: (n_samples, n_outputs) 的目标
        """
        losses = []
        for epoch in range(epochs):
            total_loss = 0.0
            for xi, yi in zip(X, y):
                # 前向
                output = self.forward(xi)
                loss = 0.5 * np.sum((yi - output) ** 2)
                total_loss += loss

                # 反向
                dW_list, db_list = self.backward(yi)

                # 更新
                self.update(dW_list, db_list, lr)

            avg_loss = total_loss / len(X)
            losses.append(avg_loss)

            if (epoch + 1) % print_every == 0:
                print(f"Epoch {epoch+1:>5d}, Loss: {avg_loss:.6f}")

        return losses


# ============================================================
# 测试：XOR 问题（线性不可分，必须多层网络才能解）
# ============================================================
print("=" * 60)
print("XOR 问题训练")
print("=" * 60)

X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]], dtype=float)
y = np.array([[0], [1], [1], [0]], dtype=float)

nn = NeuralNetwork([2, 4, 1])  # 2 输入 → 4 隐藏 → 1 输出
losses = nn.train(X, y, epochs=5000, lr=1.0, print_every=1000)

# 测试结果
print("\n预测结果:")
for xi, yi in zip(X, y):
    pred = nn.forward(xi)
    print(f"  输入: {xi} → 预测: {pred[0]:.4f}, 目标: {yi[0]:.1f}")
```

### 示例 4：观察梯度消失现象

```python
import numpy as np

# ============================================================
# 梯度消失实验：对比 Sigmoid vs ReLU 在深层网络中的梯度表现
# ============================================================

def build_deep_network(n_layers, hidden_size, activation='sigmoid'):
    """构建一个深层网络并计算各层梯度的范数。"""
    np.random.seed(42)

    # 初始化权重
    weights = []
    for i in range(n_layers):
        # 简单的随机初始化（故意不用 Xavier，以更明显地展示梯度消失）
        W = np.random.randn(hidden_size, hidden_size) * 0.5
        weights.append(W)

    # 前向传播，保存每层的 z 和 a
    x = np.random.randn(hidden_size)
    z_list = []
    a_list = [x]
    a = x

    for W in weights:
        z = W @ a
        if activation == 'sigmoid':
            a = 1.0 / (1.0 + np.exp(-np.clip(z, -500, 500)))
        elif activation == 'relu':
            a = np.maximum(0, z)
        z_list.append(z)
        a_list.append(a)

    # 假设损失对最后一层输出的梯度为全 1（简化）
    grad = np.ones(hidden_size)

    # 反向传播，记录每层梯度的范数
    grad_norms = []
    for i in reversed(range(n_layers)):
        z = z_list[i]
        if activation == 'sigmoid':
            s = 1.0 / (1.0 + np.exp(-np.clip(z, -500, 500)))
            f_prime = s * (1.0 - s)
        elif activation == 'relu':
            f_prime = (z > 0).astype(float)

        grad = grad * f_prime            # 穿过激活函数
        grad_norms.insert(0, np.linalg.norm(grad))
        grad = weights[i].T @ grad       # 穿过线性变换

    return grad_norms

# 测试不同深度

    print(f"  Sigmoid 各层梯度范数:")
    for i, norm in enumerate(norms_sigmoid):
        bar = '█' * max(1, int(norm * 50))

    # ReLU 激活

    print(f"  ReLU 各层梯度范数:")
    for i, norm in enumerate(norms_relu):
        bar = '█' * max(1, min(50, int(norm * 5)))
        print(f"    第 {i+1:>2d} 层: {norm:.6e}  {bar}")

    # 对比
    if len(norms_sigmoid) > 1 and norms_sigmoid[-1] > 0:
        ratio = norms_sigmoid[0] / norms_sigmoid[-1]
        print(f"  Sigmoid 第1层/最后层梯度比: {ratio:.2e}")
```

## 工程师视角

### PyTorch 的 autograd：你只需要 `loss.backward()`

在实际工程中，你**几乎永远不需要手写反向传播**。PyTorch 的自动微分引擎会在前向传播时自动构建计算图，调用 `loss.backward()` 时自动执行反向传播，然后优化器 `optimizer.step()` 完成参数更新。

```python
import torch
import torch.nn as nn
import torch.optim as optim

# ============================================================
# 实际工程中的训练循环（标准模板）
# ============================================================

# 1. 定义模型
model = nn.Sequential(
    nn.Linear(784, 256),
    nn.ReLU(),
    nn.Linear(256, 128),
    nn.ReLU(),
    nn.Linear(128, 10),
)

# 2. 定义损失函数和优化器
criterion = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=1e-3)

# 3. 训练循环
# for epoch in range(num_epochs):
#     for batch_x, batch_y in dataloader:
#
#         # 前向传播：模型自动构建计算图
#         predictions = model(batch_x)
#         loss = criterion(predictions, batch_y)
#
#         # 反向传播：一行搞定所有梯度计算
#         optimizer.zero_grad()   # 清除上一步的梯度（重要！）
#         loss.backward()         # 自动反向传播，计算所有参数的梯度

# (a) 为什么要 zero_grad()？

# - torch.no_grad()：推理时不需要梯度，节省内存

with torch.no_grad():
    # 推理模式：不构建计算图，内存占用减半
    test_output = model(x_demo)
    print(f"\n推理输出 shape: {test_output.shape}")
    # test_output 没有 grad_fn，无法 backward
```

### 计算图：动态图 vs 静态图

| 特性 | PyTorch（动态图） | TensorFlow 1.x（静态图） |
|------|-------------------|--------------------------|
| 图的构建时机 | **每次前向传播时实时构建** | 先定义图结构，再喂数据运行 |
| 控制流 | 直接用 Python if/for | 需要用 tf.cond, tf.while_loop |
| 调试 | 随时 print、pdb | 需要 tf.Print 或 Session.run |
| 性能优化 | JIT 编译（torch.compile） | 全局优化（XLA） |
| 心智模型 | "写 Python = 写模型" | "先画蓝图，再施工" |

**为什么动态图胜出？** 对工程师来说，能直接用 Python 的 if/for/print 来写模型逻辑，调试成本极低。这就是 PyTorch 在研究界和工业界全面胜出的关键原因。TensorFlow 2.x 也转向了 Eager Mode（本质上就是动态图）。

```python
import torch
import torch.nn as nn

# ============================================================
# 动态图的威力：模型结构可以随输入变化
# ============================================================

class DynamicNet(nn.Module):
    """
    一个根据输入动态决定计算路径的网络。
    这在静态图框架中非常难实现。
    """
    def __init__(self):
        super().__init__()
        self.linear1 = nn.Linear(10, 10)
        self.linear2 = nn.Linear(10, 10)
        self.linear3 = nn.Linear(10, 10)
        self.output = nn.Linear(10, 1)

    def forward(self, x):
        # 普通的 Python if：根据输入的范数动态选择路径
        h = torch.relu(self.linear1(x))

        if h.norm() > 1.0:
            # 输入"能量"大时，多加一层处理
            h = torch.relu(self.linear2(h))
            h = torch.relu(self.linear3(h))
        else:
            # 输入"能量"小时，只用一层
            h = torch.relu(self.linear2(h))

        return self.output(h)

net = DynamicNet()
x1 = torch.randn(1, 10) * 0.1  # 小输入
x2 = torch.randn(1, 10) * 10.0  # 大输入

y1 = net(x1)
y2 = net(x2)

# 两次前向传播走了不同的路径，但反向传播都能正确计算梯度！
y1.backward()
print(f"小输入：linear3 的梯度是否为 None？{net.linear3.weight.grad is None}")

net.zero_grad()
y2.backward()
print(f"大输入：linear3 的梯度是否为 None？{net.linear3.weight.grad is None}")
# 大输入走了 3 层路径，所以 linear3 有梯度
```

### 梯度检查（Gradient Checking）

梯度检查是调试反向传播实现的标准手段：**用数值方法（有限差分）逐个参数计算梯度，与解析梯度（反向传播的结果）对比**。如果差异在 $10^{-5}$ 以内，说明反向传播实现正确。

这个技巧在以下场景特别有用：
- 你自定义了一个新的层或损失函数，需要验证导数是否写对了
- 模型训练 loss 不降，怀疑梯度计算有 bug
- 教学目的，验证手算结果

```python
import torch
import torch.nn as nn

# ============================================================
# 梯度检查：数值梯度 vs 解析梯度
# ============================================================

def gradient_check(model, x, y, criterion, eps=1e-5):
    """
    对模型的每个参数做数值梯度检查。

    原理：
      数值梯度 ≈ [L(θ+ε) - L(θ-ε)] / (2ε)   （中心差分）
      解析梯度 = loss.backward() 得到的 .grad

    如果两者的相对差异 < 1e-5，说明反向传播正确。
    """
    # 先用反向传播算解析梯度
    model.zero_grad()
    output = model(x)
    loss = criterion(output, y)
    loss.backward()

    print(f"Loss = {loss.item():.6f}\n")

    for name, param in model.named_parameters():
        if param.grad is None:
            continue

        analytical_grad = param.grad.clone()
        numerical_grad = torch.zeros_like(param)

        # 对每个参数元素做数值求导
        param_flat = param.data.view(-1)
        grad_flat = numerical_grad.view(-1)

        for i in range(min(len(param_flat), 10)):  # 只检查前 10 个元素（大参数太慢）
            old_val = param_flat[i].item()

            # L(θ + ε)
            param_flat[i] = old_val + eps
            with torch.no_grad():
                loss_plus = criterion(model(x), y).item()

            # L(θ - ε)
            param_flat[i] = old_val - eps
            with torch.no_grad():
                loss_minus = criterion(model(x), y).item()

            # 还原
            param_flat[i] = old_val

            # 数值梯度
            grad_flat[i] = (loss_plus - loss_minus) / (2 * eps)

        # 比较
        analyt_flat = analytical_grad.view(-1)[:10]
        numer_flat = numerical_grad.view(-1)[:10]

        # 相对误差
        diff = torch.abs(analyt_flat - numer_flat)
        denom = torch.abs(analyt_flat) + torch.abs(numer_flat) + 1e-8
        rel_error = (diff / denom).max().item()

        status = "PASS" if rel_error < 1e-4 else "FAIL"
        print(f"  {name:>15s}: 最大相对误差 = {rel_error:.2e}  [{status}]")

# 测试
model = nn.Sequential(
    nn.Linear(3, 4),
    nn.Sigmoid(),
    nn.Linear(4, 2),
    nn.Sigmoid(),
)

x = torch.randn(2, 3)
y = torch.randn(2, 2)
criterion = nn.MSELoss()

print("梯度检查结果:")
gradient_check(model, x, y, criterion)
```

### 梯度裁剪（Gradient Clipping）

梯度爆炸在 RNN/LSTM 和 Transformer 的训练中非常常见。**梯度裁剪**是最简单有效的防御手段：给梯度设一个上界，如果梯度范数超过阈值，就等比例缩放到阈值以内。

```python
import torch
import torch.nn as nn

# ============================================================
# 梯度裁剪：防止梯度爆炸
# ============================================================

model = nn.Sequential(
    nn.Linear(10, 10),
    nn.ReLU(),
    nn.Linear(10, 10),
    nn.ReLU(),
    nn.Linear(10, 1),
)

optimizer = torch.optim.Adam(model.parameters(), lr=1e-3)

# 模拟一次训练步骤
x = torch.randn(4, 10)
y = torch.randn(4, 1)

loss = nn.MSELoss()(model(x), y)
loss.backward()

# 查看裁剪前的梯度范数
total_norm_before = torch.nn.utils.clip_grad_norm_(model.parameters(),
                                                     float('inf'))
print(f"裁剪前梯度总范数: {total_norm_before:.4f}")

# --- 方法 1：按范数裁剪（最常用）---
# 如果梯度的 L2 范数 > max_norm，则按比例缩放
# 公式：grad = grad * (max_norm / grad_norm)   when grad_norm > max_norm
optimizer.zero_grad()
loss = nn.MSELoss()(model(x), y)
loss.backward()

max_norm = 1.0
total_norm = torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm)
print(f"\n按范数裁剪 (max_norm={max_norm}):")
print(f"  原始梯度范数: {total_norm:.4f}")
# 裁剪后，总范数不超过 max_norm
clipped_norm = sum(p.grad.norm()**2 for p in model.parameters()
                   if p.grad is not None) ** 0.5
print(f"  裁剪后梯度范数: {clipped_norm:.4f}")

# --- 方法 2：按值裁剪 ---
# 直接把每个梯度元素限制在 [-clip_value, clip_value] 范围内
optimizer.zero_grad()
loss = nn.MSELoss()(model(x), y)
loss.backward()

clip_value = 0.5
torch.nn.utils.clip_grad_value_(model.parameters(), clip_value)
print(f"\n按值裁剪 (clip_value={clip_value}):")
for name, param in model.named_parameters():
    if param.grad is not None:
        print(f"  {name}: 梯度范围 [{param.grad.min():.4f}, {param.grad.max():.4f}]")

# --- 实际训练中的典型用法 ---
print("\n" + "=" * 60)
print("工程实践：典型训练循环中的梯度裁剪")
print("=" * 60)
print("""
for batch_x, batch_y in dataloader:
    optimizer.zero_grad()
    output = model(batch_x)
    loss = criterion(output, batch_y)
    loss.backward()

    # 在 optimizer.step() 之前裁剪梯度
    torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)

    optimizer.step()
""")

# --- 各类模型的推荐裁剪值 ---
print("常见模型的梯度裁剪设置:")
print("  Transformer (NLP):  max_norm = 1.0")
print("  GPT 系列:           max_norm = 1.0")
print("  LSTM/GRU:           max_norm = 5.0")
print("  Diffusion Models:   通常不需要裁剪（U-Net 较稳定）")
print("  ViT (Vision):       max_norm = 1.0 ~ 5.0")
```

### 常见陷阱与调试技巧

| 症状 | 可能原因 | 排查方法 |
|------|----------|----------|
| loss 为 NaN | 梯度爆炸、学习率过大、数值溢出 | 降低 lr，加梯度裁剪，检查 log(0) |
| loss 不下降 | 忘记 zero_grad()、学习率过小、模型太简单 | 打印梯度范数，检查是否为 0 |
| loss 下降后又上升 | 学习率过大（震荡）、数据有问题 | 用 lr scheduler，检查数据加载 |
| 前几层梯度为 0 | 梯度消失、死亡 ReLU | 检查激活函数，用 BatchNorm/残差连接 |
| 训练很慢 | 梯度消失、初始化不当 | 换 Xavier/He 初始化，用 Adam |

```python
import torch
import torch.nn as nn

# ============================================================
# 梯度监控工具：训练时随时查看梯度健康状况
# ============================================================

def print_grad_stats(model, step=None):
    """打印模型各层的梯度统计信息，用于调试。"""
    prefix = f"[Step {step}] " if step is not None else ""
    print(f"{prefix}梯度统计:")
    for name, param in model.named_parameters():
        if param.grad is not None:
            grad = param.grad
            print(f"  {name:>25s} | "
                  f"mean: {grad.mean():>10.2e} | "
                  f"std: {grad.std():>10.2e} | "
                  f"max: {grad.abs().max():>10.2e} | "
                  f"min: {grad.abs().min():>10.2e}")
        else:
            print(f"  {name:>25s} | grad = None (未参与计算)")

# 使用示例
model = nn.Sequential(
    nn.Linear(10, 64), nn.ReLU(),
    nn.Linear(64, 64), nn.ReLU(),
    nn.Linear(64, 1),
)

x = torch.randn(8, 10)
y = torch.randn(8, 1)
loss = nn.MSELoss()(model(x), y)
loss.backward()

print_grad_stats(model, step=0)
```

## 本章小结

| 概念 | 一句话总结 |
|------|-----------|
| **计算图** | 把前向传播拆成一步步的运算节点，构成有向无环图 |
| **链式法则** | 复合函数的导数 = 逐层局部导数的连乘 |
| **反向传播** | 从损失出发，沿计算图反向传递梯度，一次性算出所有参数的偏导 |
| **误差项 $\delta$** | 某层线性输出 $z$ 对损失的偏导，是梯度"回传"的载体 |
| **梯度消失** | 深层网络中，靠近输入的层梯度指数级衰减（Sigmoid 每层最多保留 25%） |
| **梯度爆炸** | 深层网络中，梯度指数级增长，导致参数更新过大、训练不稳定 |
| **ReLU** | 缓解梯度消失：z>0 时导数为 1，梯度无损传递 |
| **梯度裁剪** | 给梯度范数设上界，防止梯度爆炸 |
| **autograd** | PyTorch 自动微分引擎，`loss.backward()` 一行搞定所有梯度 |
| **梯度检查** | 用数值差分验证解析梯度，调试自定义层的标准手段 |
