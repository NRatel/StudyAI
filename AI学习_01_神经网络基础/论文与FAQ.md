## 一、关键论文脉络

| 年份 | 论文/方法 | 为什么重要 |
|------|-----------|------------|
| 1958 | Rosenblatt — Perceptron | 提出可学习权重的人工神经元，神经网络正式起步 |
| 1986 | Rumelhart et al. — Backpropagation | 让多层网络可以高效训练 |
| 1989 | Hornik — Universal Approximation | 说明神经网络有强表达能力，但不保证容易训练 |
| 2014 | Dropout | 用随机丢弃神经元缓解过拟合 |
| 2015 | Kaiming Initialization | 让 ReLU 深层网络更容易稳定启动训练 |
| 2015 | Batch Normalization | 让训练更稳定，也带来一定正则化效果 |
| 2015 | Adam | 结合动量和自适应学习率，成为常见优化器 |
| 2016 | GELU | 后来在 Transformer 中广泛使用的平滑激活函数 |
| 2019 | AdamW | 更清晰地处理权重衰减，现代大模型常用 |

阅读顺序建议：

1. 先理解反向传播和梯度下降。
2. 再理解过拟合、正则化、初始化。
3. 最后再看 Adam、AdamW、GELU 这些现代组件。

不建议初学时逐篇论文啃细节。先把正文里的概念连起来，论文只作为历史坐标。

## 二、常见误区与 FAQ

### Q1：神经网络是在模拟人脑吗？

不是严格模拟。

“神经网络”这个名字确实来自生物启发：输入信号、加权求和、超过阈值后输出。但现代人工神经网络更准确地说是一类数学函数模型。

可以这样理解：

```text
生物大脑 → 灵感来源
人工神经网络 → 可训练的数学函数
```

不要用生物神经科学去强行解释每个深度学习组件。比如反向传播需要明确的损失函数和梯度链路，这和真实大脑的学习机制并不等价。

### Q2：层数越多、参数越多，模型一定越好吗？

不一定。

模型变大通常会增加表达能力，但也带来三个问题：

- **过拟合**：数据不够时，模型可能记住噪声。
- **优化困难**：网络太深时，梯度可能消失、爆炸或训练不稳定。
- **成本增加**：训练和推理都更慢、更贵。

更准确的说法是：模型规模、数据量、训练方法要匹配。只堆参数不是万能药。

### Q3：学习率大一点是不是训练更快？

学习率决定每次参数更新走多远。

| 情况 | 现象 |
|------|------|
| 学习率太大 | 损失震荡、发散，甚至出现 NaN |
| 学习率太小 | 收敛很慢，训练很久仍学不到 |
| 学习率合适 | 损失整体下降，训练稳定 |

直觉上像下山：步子太大容易跨过山谷，步子太小走得太慢。Adam 能缓解一些问题，但不能让学习率不重要。

### Q4：过拟合和欠拟合怎么区分？

看训练集和验证集。

| 状态 | 训练集表现 | 验证集表现 | 含义 |
|------|------------|------------|------|
| 欠拟合 | 差 | 差 | 模型还没学会基本规律 |
| 过拟合 | 好 | 差 | 模型记住训练集细节，泛化差 |
| 合适 | 好 | 好 | 学到可迁移规律 |

常见处理：

- 欠拟合：增加模型能力、训练更久、检查输入特征或数据质量。
- 过拟合：增加数据、数据增强、正则化、Dropout、早停、降低模型复杂度。

### Q5：为什么激活函数不能都用线性函数？

因为线性函数叠多少层还是线性函数。

没有非线性激活时：

```text
多层网络 = 一个更大的线性变换
```

它仍然只能学习线性边界，无法表达复杂模式。激活函数的核心价值就是打破这个限制。

### Q6：ReLU 有死神经元问题，为什么还常用？

因为它简单、稳定、训练快。

ReLU 的正半轴梯度为 1，能显著缓解 Sigmoid/Tanh 在深层网络中的梯度消失问题。死神经元确实存在，但通常可以通过合理初始化、学习率控制，或换成 Leaky ReLU/GELU 来缓解。

所以 ReLU 不是完美函数，而是一个非常实用的默认选择。

### Q7：损失函数和评估指标是一回事吗？

不是。

| 概念 | 作用 |
|------|------|
| 损失函数 | 训练时优化器直接最小化的目标，通常需要可微 |
| 评估指标 | 人真正关心的效果度量，不一定可微 |

例如分类任务可以用交叉熵训练，但用准确率、F1、召回率评估。损失低通常是好信号，但最终仍要看任务指标。

### Q8：为什么不能把权重全初始化为 0？

因为会造成**对称性问题**。

如果同一层所有神经元权重都一样，它们会得到一样的输出、一样的梯度、一样的更新。训练多久都像复制出来的同一个神经元，学不到不同特征。

随机初始化的作用不是“随便猜”，而是打破对称性，让不同神经元有机会学不同东西。
