激活函数

前置知识：感知机与多层网络
本节目标：理解激活函数为什么必要，以及 ReLU、Sigmoid、Tanh、GELU、SiLU、Softmax 各自大概适合什么场景。

1. 激活函数解决什么问题

神经网络每一层都会先做一件很朴素的事：把输入乘上权重，再加上偏置。

$$z = Wx + b$$

这叫线性变换。它很重要，但也有一个大限制：线性变换只能表达直线、平面这一类关系。

如果网络只有线性变换，没有激活函数，那么层数再多也没有本质变化：

$$W_2(W_1x + b_1) + b_2 = W'x + b'$$

换句话说：

没有激活函数：
很多层线性变换 = 一层更大的线性变换

看起来像深度网络，实际只是“套娃版直线工具”。

激活函数的作用就是在线性变换之后加一道非线性规则。它让网络拥有“拐弯能力”，可以表达曲线、复杂边界和不同特征之间的组合关系。

直白地说：

线性层负责把信息混合起来，激活函数负责让模型不再只能画直线。

2. ReLU：最常见的单向阀

ReLU 是最常见、最容易理解的激活函数之一。

$$\text{ReLU}(x)=\max(0,x)$$

它的规则只有一句话：

  • 输入是正数，就原样通过。
  • 输入是负数，就变成 0。
输入： -3  -1   0   2   5
输出：  0   0   0   2   5

它像一个单向阀：有用的正向信号可以通过，负向信号被截断。

ReLU 受欢迎，是因为它简单、计算快，而且在很多网络里训练稳定。CNN、普通 MLP 等模型经常使用它。

它也有缺点。假如某个神经元长期只收到负输入，它会一直输出 0，训练时也很难再调整回来。这种情况常被叫做死神经元。名字有点夸张，本质就是这个神经元暂时不怎么参与工作了。

3. Sigmoid：把分数压成概率

Sigmoid 会把任意输入压到 0 到 1 之间：

$$\sigma(x)=\frac{1}{1+e^{-x}}$$

所以它很适合回答这种问题：

这张图片是猫的概率是多少？

如果输出是 0.92，我们可以把它理解成模型认为“是猫”的概率很高。

Sigmoid 常见于二分类输出层。比如判断邮件是不是垃圾邮件、交易是不是欺诈、图片里有没有某个对象。

但 Sigmoid 不太适合放在很深网络的中间层。原因是它两端会变得很平：输入特别大或特别小时，输出变化很小，训练信号容易变弱。这就是常说的梯度消失风险之一。

4. Tanh：带正负方向的压缩器

Tanh 和 Sigmoid 很像，也会把输入压到一个固定范围里。不同的是，Tanh 的输出范围是 -1 到 1：

$$\tanh(x)=\frac{e^x-e^{-x}}{e^x+e^{-x}}$$

它可以表达“正方向”和“负方向”：

  • 接近 1：强烈正向。
  • 接近 -1：强烈负向。
  • 接近 0：影响较弱。

Tanh 曾经在 RNN、LSTM 等序列模型里很常见，用来表示中间状态。它的问题和 Sigmoid 类似：输入太大或太小时也会变平，深层训练时可能让梯度变弱。

5. GELU 和 SiLU：更平滑的现代闸门

GELU 和 SiLU 可以理解成“平滑版 ReLU”。

ReLU 像硬开关：负数直接切掉，正数直接放行。

GELU 和 SiLU 更像软开关：不是一刀切，而是根据输入大小平滑地决定放行多少。

函数直觉常见位置
GELU根据输入大小平滑决定通过比例BERT、GPT 等 Transformer
SiLU / Swish用一个平滑门控调节信号LLaMA 等现代模型

学习到这里，不需要记住它们复杂的展开式。只要知道：很多现代大模型在隐藏层里更喜欢这类平滑激活函数，因为它们通常比硬切换更适合大规模训练。

6. Softmax：把一组分数变成一组概率

Softmax 和前面的激活函数不太一样。ReLU、Sigmoid、Tanh 通常是对每个数单独处理；Softmax 是把一整组分数一起处理。

比如模型给三个类别打了分：

猫：8.0
狗：3.0
车：1.0

这些只是分数，还不是概率。Softmax 会把它们转换成一组加起来等于 1 的概率：

猫：很高
狗：较低
车：更低

公式是：

$$\text{softmax}(z_i)=\frac{e^{z_i}}{\sum_j e^{z_j}}$$

它有两个关键性质：

  • 每个输出都大于 0。
  • 所有输出加起来等于 1。

所以 Softmax 常用于两个地方：

  • 多分类输出层：让模型从多个类别里选一个。
  • 注意力机制：把相关性分数变成注意力权重，也就是决定“更该看谁”。

7. 怎么选择激活函数

初学时不用把选择规则想得太复杂。先抓住下面这张表：

场景常见选择为什么
普通隐藏层ReLU简单、快、效果稳
Transformer 隐藏层GELU 或 SiLU平滑，适合现代大模型
RNN/LSTM 中间状态Tanh / Sigmoid需要有界范围和门控
二分类输出层Sigmoid输出一个 0 到 1 的概率
多分类输出层Softmax输出一组概率分布
回归输出层通常不加激活允许输出任意实数

最重要的判断是：

隐藏层：主要负责提取特征，常用 ReLU / GELU / SiLU
输出层：要看答案需要什么形式，是概率、类别，还是连续数值

激活函数不是装饰品。没有它，多层网络会退化成一层线性模型；有了它，网络才真正拥有表达复杂关系的能力。
