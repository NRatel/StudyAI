  1. жіЁж„ҸеҠӣеұӮзҡ„иҫ“еҮәжҖҺд№Ҳдј з»ҷдёӢдёҖеұӮпјҹзӣҙжҺҘе ҶеҸ дјҡдёҚдјҡеҜјиҮҙи®ӯз»ғдёҚзЁіе®ҡпјҹ
  2. еҸӘжңүжіЁж„ҸеҠӣеӨҹеҗ—пјҹиҝҳйңҖиҰҒд»Җд№Ҳе…¶д»–з»„д»¶пјҹ

з”Ёж•°еӯҰиЎЁзӨәпјҲPost-LN еҪўејҸпјҢеҚіеҺҹе§Ӣи®әж–Үзҡ„еҶҷжі•пјүпјҡ

е…¶дёӯ $\text{MultiHeadAttention}(Q, K, V)$ зҡ„дёүдёӘеҸӮж•°йғҪжҳҜ $x$ иҮӘиә«пјҢеӣ жӯӨжҳҜиҮӘжіЁж„ҸеҠӣгҖӮ

ж®Ӣе·®иҝһжҺҘпјҲResidual Connectionпјү

ж®Ӣе·®иҝһжҺҘзҡ„е…¬ејҸпјҡ$\text{output} = x + F(x)$пјҢеҚі"и·іиҝҮ"еӯҗеұӮпјҢзӣҙжҺҘжҠҠиҫ“е…ҘеҠ еҲ°еӯҗеұӮзҡ„иҫ“еҮәдёҠгҖӮ

дёәд»Җд№ҲйңҖиҰҒж®Ӣе·®иҝһжҺҘпјҹ жІЎжңүж®Ӣе·®иҝһжҺҘж—¶пјҢжўҜеәҰд»Һз¬¬ $L$ еұӮдј еҲ°з¬¬ $1$ еұӮйңҖиҰҒз»ҸиҝҮжүҖжңүдёӯй—ҙеұӮзҡ„иҝһд№ҳгҖӮеҰӮжһңжҹҗдәӣеұӮзҡ„жўҜеәҰе°ҸдәҺ 1пјҢиҝһд№ҳеҗҺжўҜеәҰдјҡжҢҮж•°зә§иЎ°еҮҸгҖӮжңүдәҶж®Ӣе·®иҝһжҺҘеҗҺпјҡ

$$x_{l+1} = x_l + F_l(x_l) \quad \Rightarrow \quad \frac{\partial x_{l+1}}{\partial x_l} = I + \frac{\partial F_l(x_l)}{\partial x_l}$$

еҚідҪҝ $\frac{\partial F_l}{\partial x_l}$ еҫҲе°ҸпјҢжўҜеәҰдёӯд»ҚеҢ…еҗ« identity йЎ№ $I$пјҢдёәжўҜеәҰжҸҗдҫӣдәҶзӣҙжҺҘдј ж’ӯи·Ҝеҫ„пјҢйҖҡеёёжҳҫи‘—зј“и§Јж·ұеұӮзҪ‘з»ңзҡ„жўҜеәҰиЎ°еҮҸй—®йўҳпјҲжіЁж„ҸиҝҷдёҚжҳҜз»қеҜ№зҡ„ж•°еӯҰдҝқиҜҒпјҢиҖҢжҳҜе®һи·өдёӯйқһеёёжңүж•Ҳзҡ„жңәеҲ¶пјүгҖӮ6 еұӮз”ҡиҮіеҮ еҚҒеұӮзҡ„ Transformer иғҪзЁіе®ҡи®ӯз»ғпјҢж®Ӣе·®иҝһжҺҘеҠҹдёҚеҸҜжІЎгҖӮ

Layer Normalization

Layer Norm еҜ№жҜҸдёӘж ·жң¬еңЁзү№еҫҒз»ҙеәҰдёҠеҒҡеҪ’дёҖеҢ–пјҡ

$$\text{LayerNorm}(x) = \gamma \odot \frac{x - \mu}{\sqrt{\sigma^2 + \epsilon}} + \beta$$

е…¶дёӯ $\mu, \sigma^2$ еңЁзү№еҫҒз»ҙеәҰдёҠи®Ўз®—пјҢ$\gamma, \beta$ жҳҜеҸҜеӯҰд№ еҸӮж•°гҖӮ

зү№жҖ§Batch NormLayer Norm
еҪ’дёҖеҢ–з»ҙеәҰbatch з»ҙеәҰпјҲи·Ёж ·жң¬пјүзү№еҫҒз»ҙеәҰпјҲжҜҸдёӘж ·жң¬зӢ¬з«Ӣпјү
дҫқиө– batch еӨ§е°ҸжҳҜпјҲе°Ҹ batch дёҚзЁіе®ҡпјүеҗҰ
еәҸеҲ—й•ҝеәҰеҸҜеҸҳдёҚж–№дҫҝеӨ©з„¶ж”ҜжҢҒ

Layer Norm еңЁ Transformer дёӯзҡ„дҪҚзҪ®жңүдёӨз§ҚжөҒжҙҫпјҡ

  • Post-LNпјҲеҺҹе§Ӣи®әж–Үпјүпјҡ$\text{LayerNorm}(x + \text{Sublayer}(x))$
  • Pre-LNпјҲзҺ°д»ЈжЁЎеһӢпјүпјҡ$x + \text{Sublayer}(\text{LayerNorm}(x))$

еүҚйҰҲзҪ‘з»ңпјҲFFNпјү

жҜҸдёӘзј–з ҒеҷЁеұӮдёӯзҡ„ FFN жҳҜдёҖдёӘдҪҚзҪ®зӢ¬з«Ӣзҡ„дёӨеұӮе…ЁиҝһжҺҘзҪ‘з»ңпјҡ

$$\text{FFN}(x) = W_2 \cdot \text{Activation}(W_1 x + b_1) + b_2$$
  • $W_1 \in \mathbb{R}^{d_{ff} \times d_{model}}$пјҢ$W_2 \in \mathbb{R}^{d_{model} \times d_{ff}}$
  • $d_{ff}$ йҖҡеёёжҳҜ $d_{model}$ зҡ„ 4 еҖҚпјҲеҺҹе§Ӣи®әж–Үпјҡ$d_{model}=512, d_{ff}=2048$пјү
  • жҝҖжҙ»еҮҪж•°зҡ„жј”еҢ–пјҡReLUпјҲеҺҹе§ӢпјүвҶ’ GELUпјҲGPT-2/BERTпјүвҶ’ SwiGLUпјҲLLaMA/PaLMпјү

FFN зҡ„дҪңз”ЁпјҡжіЁж„ҸеҠӣеұӮиҙҹиҙЈ"дҪҚзҪ®д№Ӣй—ҙзҡ„дҝЎжҒҜдәӨжҚў"пјҢFFN иҙҹиҙЈ"жҜҸдёӘдҪҚзҪ®еҶ…йғЁзҡ„йқһзәҝжҖ§еҸҳжҚў"гҖӮжңүз ”з©¶иЎЁжҳҺпјҢFFN зҡ„еҸӮж•°дёӯеӯҳеӮЁдәҶеӨ§йҮҸзҡ„дәӢе®һзҹҘиҜҶпјҲзұ»дјјдәҺй”®еҖјеӯҳеӮЁпјүгҖӮ

дәӨеҸүжіЁж„ҸеҠӣпјҲCross-Attentionпјү

дәӨеҸүжіЁж„ҸеҠӣжҳҜи§Јз ҒеҷЁ"иҜ»еҸ–"зј–з ҒеҷЁдҝЎжҒҜзҡ„йҖҡйҒ“гҖӮе…ій”®еҢәеҲ«еңЁдәҺ QгҖҒKгҖҒV зҡ„жқҘжәҗпјҡ

еҸӮж•°жқҘжәҗеҗ«д№ү
Qи§Јз ҒеҷЁеҪ“еүҚеұӮзҡ„иЎЁзӨә"жҲ‘жғіжҹҘиҜўд»Җд№Ҳ"
Kзј–з ҒеҷЁзҡ„жңҖз»Ҳиҫ“еҮә"иҫ“е…ҘеәҸеҲ—жңүд»Җд№ҲеҸҜжҹҘзҡ„"
Vзј–з ҒеҷЁзҡ„жңҖз»Ҳиҫ“еҮә"жҹҘеҲ°еҗҺеҸ–д»Җд№ҲеҖј"

е®Ңж•ҙж•°жҚ®жөҒ

зј–з ҒеҷЁдҫ§пјҡ

иҫ“е…Ҙ token IDs вҶ’ Embedding (m, 512) вҶ’ + дҪҚзҪ®зј–з Ғ
  вҶ’ Encoder Layer 1: Self-Attention вҶ’ Add&Norm вҶ’ FFN вҶ’ Add&Norm
  вҶ’ Encoder Layer 2~6: (зӣёеҗҢз»“жһ„)
  вҶ’ зј–з ҒеҷЁиҫ“еҮә: (m, 512)

и§Јз ҒеҷЁдҫ§пјҡ

зӣ®ж Ү token IDsпјҲеҸіз§»дёҖдҪҚпјүвҶ’ Embedding (n, 512) вҶ’ + дҪҚзҪ®зј–з Ғ
  вҶ’ Decoder Layer 1:
      Masked Self-Attention вҶ’ Add&Norm
      Cross-Attention(Q=иҮӘиә«, K=V=зј–з ҒеҷЁиҫ“еҮә) вҶ’ Add&Norm
      FFN вҶ’ Add&Norm
  вҶ’ Decoder Layer 2~6: (зӣёеҗҢз»“жһ„)
  вҶ’ Linear (n, vocab_size) вҶ’ Softmax вҶ’ жҰӮзҺҮеҲҶеёғ

жҰӮеҝөдёҖеҸҘиҜқжҖ»з»“

и§Јз ҒеҷЁеұӮMasked Self-Attention + Cross-Attention + FFN
ж®Ӣе·®иҝһжҺҘ$x + F(x)$пјҢдҝқиҜҒжўҜеәҰиғҪи·ЁеұӮзӣҙжҺҘжөҒеҠЁ
Layer NormеңЁзү№еҫҒз»ҙеәҰдёҠеҪ’дёҖеҢ–пјҢзЁіе®ҡи®ӯз»ғ
FFNдҪҚзҪ®зӢ¬з«Ӣзҡ„дёӨеұӮе…ЁиҝһжҺҘпјҢйҡҗи—Ҹз»ҙеәҰйҖҡеёё $4 \times d_{model}$
еӣ жһңжҺ©з ҒдёҠдёүи§’и®ҫдёә $-\infty$пјҢйҳІжӯўи§Јз ҒеҷЁеҒ·зңӢжңӘжқҘ token
дәӨеҸүжіЁж„ҸеҠӣQ жқҘиҮӘи§Јз ҒеҷЁпјҢK/V жқҘиҮӘзј–з ҒеҷЁпјҢжҳҜдҝЎжҒҜдј йҖ’зҡ„йҖҡйҒ“
Pre-LNе…Ҳ Norm еҗҺеӯҗеұӮпјҢзҺ°д»Ј LLM зҡ„ж ҮеҮҶйҖүжӢ©
е№¶иЎҢдјҳеҠҝSelf-Attention дёҚдҫқиө–ж—¶й—ҙжӯҘйЎәеәҸпјҢжүҖжңүдҪҚзҪ®дёҖж¬Ўз®—еҮә

дёӢдёҖз« пјҡдҪҚзҪ®зј–з ҒвҖ”вҖ”Transformer жІЎжңү RNN зҡ„йҖ’еҪ’з»“жһ„пјҢеҰӮдҪ•зҹҘйҒ“ token зҡ„дҪҚзҪ®пјҹжӯЈејҰзј–з ҒгҖҒеҸҜеӯҰд№ зј–з ҒгҖҒRoPE еҗ„жңүд»Җд№ҲдјҳеҠЈпјҹ
