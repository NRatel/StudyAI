# 论文与 FAQ

这个模块的论文和框架很多。了解阶段先抓住：Agent 是行动循环，工具让模型连接外部世界，状态管理让任务不断线。

| 主题 | 可以关注什么 |
|------|--------------|
| ReAct | 推理和行动如何交替 |

| Multi-Agent Systems | 多角色协作何时有价值 |

## FAQ

### Agent 是不是就是加了工具的聊天机器人？

不完全是。工具调用只是部分能力，Agent 还需要目标、状态、计划、反馈和行动循环。

### 多 Agent 一定比单 Agent 强吗？

不一定。任务简单时，多 Agent 可能只会增加成本和混乱。

### 记忆越多越好吗？

不是。错误、过期、无关记忆会干扰模型，还可能带来隐私风险。

### MCP 是模型吗？

不是。它更像工具和数据源接入协议，帮助模型更规范地使用外部能力。
