# 12 Agent 与工具使用

Agent 的重点不是“会聊天”，而是能围绕目标观察环境、制定计划、调用工具、根据结果继续行动。

## 先抓住这条线

```text
目标 → 观察 → 计划 → 调工具 → 看结果 → 更新状态 → 继续
```
