# 04 预训练语言模型演进

这个模块讲语言模型如何从“给词编号”走到“先大规模预训练，再适配任务”。

{{img:ch04_00_language_model_evolution_map}}

## 先抓住这条线

```text
One-hot：词只是编号
Embedding：词有语义坐标
上下文表示：同一个词在不同句子里含义不同
BERT：更擅长理解
GPT：更擅长生成
微调：把通用模型适配到具体任务
```
