# AI 及大模型原理系统学习

从神经网络基础到 Transformer、GPT、扩散模型，再到多模态、Agent、推理等前沿方向的**自包含学习教材**。

面向有编程经验但 AI 原理了解较少的学习者，目标是系统理解 AI 与大模型的核心原理。

## 特点

- **自包含**：每份文档本身就是教材，读完即掌握，不是"推荐去哪里学"的指引
- **概念优先**：每个知识点先从具体问题切入，再讲清准确概念、核心机制和常见误区
- **分层学习**：提供核心快速、完整模型原理、完整大模型系统和全量学习路径，按目标选择，不必一上来硬刷全部内容
- **代码附录化**：代码、配置和短伪代码示例统一收纳到独立附录，主线正文不被代码打断
- **内容时效**：前沿内容截至 2026 年 4 月

## 学习路径

不必把全部内容都当成同一条必修线。先按目标选路径，读到陌生概念再回补相关模块，这比硬刷所有前置更符合真实学习节奏。

### 路径 A：核心原理快速路径

适合想先抓住大语言模型主干的学习者：

```
01 神经网络基础
  → 03 注意力机制与 Transformer
  → 04 预训练语言模型演进
  → 05 GPT 系列深度解析
```

说明：`02 经典网络架构` 是背景桥梁，不是这条快速路径的硬前置。如果读 03 时对 CNN/RNN/Seq2Seq、编码器-解码器不熟，再回补 02。

### 路径 B：完整模型原理路径

适合希望把神经网络、语言模型、生成模型和多模态模型串成完整“模型谱系”的学习者：

```
01 神经网络基础
  → 02 经典网络架构
  → 03 注意力机制与 Transformer
  → 04 预训练语言模型演进
  → 05 GPT 系列深度解析
  → 06 生成模型与扩散模型
  → 10 多模态模型
```

说明：06 不必等到 05 之后才能开始；只要完成 02，就可以和语言模型线并行学习。

### 路径 C：完整大模型系统路径

适合想理解大模型从训练、推理、接入外部知识到工具行动的完整系统链路的学习者：

```
01 神经网络基础
  → 03 注意力机制与 Transformer
  → 04 预训练语言模型演进
  → 05 GPT 系列深度解析
  → 07 大模型训练技术
  → 08 大模型推理与部署
  → 09 RAG 与应用架构
  → 11 推理与思维链
  → 12 Agent 与工具使用
```

说明：这条路径聚焦“大模型系统”，不强制学习扩散模型和多模态。如果读 03 时背景不够，仍然建议回补 02。

### 路径 D：全量学习路径

适合希望把全部内容系统过一遍的学习者：

```
01 神经网络基础
  → 02 经典网络架构

  → 05 GPT 系列深度解析
  → 06 生成模型与扩散模型
  → 07 大模型训练技术
  → 08 大模型推理与部署
  → 09 RAG 与应用架构
  → 10 多模态模型
```

说明：`13 长文本与高效架构` 建议放最后，作为高级选读。它技术上依赖 03，但读完 05/08 后再看，会更容易理解它为什么重要。

附录：数学基础速览、代码示例集按需查阅，不占主线。

## 如何使用这套教材

1. **先读索引**：从 [AI 学习目录索引](AI学习目录索引.md) 开始，了解每个模块的内容摘要、前置依赖和推荐学习顺序。
2. **按目标选路径**：想快速理解 GPT 主线，走路径 A；想理解模型谱系，走路径 B；想理解完整大模型系统，走路径 C；想全部学完，再走路径 D。
3. **进入模块**：每个模块文件夹下的 `README.md` 是模块入口，按其中列出的子文件顺序逐篇阅读。
4. **按需查阅附录**：[数学基础速览](附录_数学基础速览.md)、[代码示例集](附录_代码示例集.md) 和 [整体知识体系思维导图](整体知识体系思维导图.md) 不占主线，遇到不熟悉的数学概念或需要代码验证时再翻阅。
5. **先理解概念**：主线正文优先服务于原理理解；代码示例是辅助材料，不是学习主线。

## 适合谁

- 有 Python 编程经验，想系统理解 AI/大模型原理的学习者
- 用过 ChatGPT/Claude 等工具，但想知道"它们是怎么工作的"
- 不追求成为研究者，但想把核心概念、技术脉络和关键原理真正讲明白

## 不适合谁

- 零编程基础的初学者（需要基本的 Python 能力）
- 追求严格数学证明的研究者（本教材重直觉、概念定义和核心原理）
