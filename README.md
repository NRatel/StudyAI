# AI 及大模型原理系统学习

从神经网络基础到 Transformer、GPT、扩散模型，再到多模态、Agent、推理等前沿方向的**概念梳理材料**。

面向有编程经验但 AI 原理基础较少的读者，目标是建立 AI 与大模型核心原理的概念地图。读完后应能看懂主要术语、技术脉络和常见误区，不需要一次吃透数学推导、论文细节或工程复现。

## 特点

- **先抓主干**：先建立概念定义、直觉解释和技术脉络，不把推导和工程细节当作主线负担
- **概念自洽**：每个知识点先从具体问题切入，再讲清准确概念、核心机制和常见误区
- **分层阅读**：提供核心快速、完整模型原理、完整大模型系统和全量路径，按目标选择，不必一上来硬刷全部内容
- **代码附录化**：代码、配置和短伪代码示例统一收纳到独立附录，主线正文不被代码打断
- **内容时效**：前沿内容截至 2026 年 4 月

## 学习路径

不必把全部内容都当成同一条必修线。先按目标选路径，读到陌生概念再回补相关模块；目标是“看懂主线”，不是把每个公式、模型名和框架 API 都背下来。

### 路径 A：核心原理快速路径

适合想先抓住大语言模型主干的学习者：

```
01 神经网络基础
  → 03 注意力机制与 Transformer
  → 04 预训练语言模型演进
  → 05 GPT 系列深度解析
```

说明：`02 经典网络架构` 是背景桥梁，不是这条快速路径的硬前置。如果读 03 时对 CNN/RNN/Seq2Seq、编码器-解码器不熟，再回补 02。

### 路径 B：完整模型原理路径

适合希望把神经网络、语言模型、生成模型和多模态模型串成完整“模型谱系”的学习者：

```
01 神经网络基础
  → 02 经典网络架构
  → 03 注意力机制与 Transformer
  → 04 预训练语言模型演进
  → 05 GPT 系列深度解析
  → 06 生成模型与扩散模型
  → 10 多模态模型
```

说明：06 不必等到 05 之后才能开始；只要完成 02，就可以和语言模型线并行学习。

### 路径 C：完整大模型系统路径

适合想理解大模型从训练、推理、接入外部知识到工具行动的完整系统链路的学习者：

```
01 神经网络基础
```

## 如何使用这套内容

1. **先读索引**：从 [AI 学习目录索引](AI学习目录索引.md) 开始，了解每个模块的内容摘要、前置依赖和推荐学习顺序。
2. **按目标选路径**：想快速了解 GPT 主线，走路径 A；想理解模型谱系，走路径 B；想理解完整大模型系统，走路径 C；想全量浏览，再走路径 D。
3. **进入模块**：每个模块文件夹下的 `README.md` 是模块入口，按其中列出的子文件顺序逐篇阅读。
4. **按需查阅附录**：[术语速查](附录_术语速查.md)、[数学基础速览](附录_数学基础速览.md)、[代码示例集](附录_代码示例集.md) 和 [整体知识体系思维导图](整体知识体系思维导图.md) 不占主线，遇到陌生术语、数学概念或需要代码验证时再翻阅。
5. **抓关系而非背细节**：优先弄清“是什么、为什么出现、解决什么问题、和前后概念什么关系”。模型排行榜、框架 API、复杂公式可以二刷时再看。

## 适合谁

- 有 Python 编程经验，想系统了解 AI/大模型原理的学习者
- 用过 ChatGPT/Claude 等工具，但想知道"它们是怎么工作的"
- 不追求成为研究者或工程专家，但想把核心概念、技术脉络和关键原理真正讲明白

## 不适合谁

- 零编程基础的初学者（需要基本的 Python 能力）
- 希望通过大量练习、项目作业来训练实操能力的学习者（本项目更偏概念了解）
- 追求严格数学证明的研究者（本项目重直觉、概念定义和核心原理）
