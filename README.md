# StudyAI — AI 及大模型原理系统学习教材

从神经网络基础到 Transformer、GPT、扩散模型，再到多模态、Agent、推理等前沿方向的**自包含学习教材**。

面向有编程经验但 AI 原理了解较少的工程师，目标达到"高级工程师"理解水平。

## 特点

- **自包含**：每份文档本身就是教材，读完即掌握，不是"推荐去哪里学"的指引
- **四段式结构**：每个知识点按「直觉与概述 → 严谨原理 → Python 代码示例 → 工程师视角」展开
- **由浅入深**：从感知机一路讲到 GPT-4、Sora、Agent，有明确的依赖关系和学习路径
- **代码驱动**：大量可运行的 Python 示例（numpy / PyTorch / HuggingFace）
- **内容时效**：前沿内容截至 2026 年 4 月

## 适合谁

- 有 Python 编程经验，想系统理解 AI/大模型原理的工程师
- 用过 ChatGPT/Claude 等工具，但想知道"它们是怎么工作的"
- 不追求成为研究者，但想达到能读懂论文、能做技术选型的水平

## 不适合谁

- 零编程基础的初学者（需要基本的 Python 能力）
- 追求严格数学证明的研究者（本教材重直觉和工程实践）
