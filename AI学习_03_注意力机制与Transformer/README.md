# 03 注意力机制与 Transformer

这个模块是理解大模型的核心。Transformer 的关键不是“结构很复杂”，而是它让文本里的每个位置都能直接查看其他位置，从而更好地建立关系。

## 先抓住这条线

```text
注意力：重点看谁
Self-Attention：一句话里的词互相看
Multi-Head：从多个角度同时看
Transformer：把注意力、前馈网络、残差和归一化堆成稳定结构
位置编码：告诉模型顺序和距离
```
