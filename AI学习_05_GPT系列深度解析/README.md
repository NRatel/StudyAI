# 05 GPT 系列深度解析

GPT 的主线其实很简单：用 Transformer Decoder 根据前文预测下一个 token。真正复杂的是，当模型、数据和训练规模不断变大后，这个简单目标会长出很强的通用能力。

## 先抓住这条线

```text
自回归生成 → 规模扩大 → 能力涌现 → 指令微调 → 偏好对齐 → 多模态、工具、推理继续扩展
```

## 读完应该能解释

- GPT 为什么叫自回归模型。
- Scaling Laws 说的是趋势，不是神话。
- 对齐不是给模型塞知识，而是校准行为。
- 新一代模型的竞争不只是参数规模。
