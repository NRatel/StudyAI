# 11 推理与思维链

这个模块讲模型怎样处理多步问题。重点不是把模型想成“真的在脑内思考”，而是理解：把复杂问题拆成中间步骤，通常能提高回答质量。

## 先抓住这条线

```text
直接回答容易跳步 → 拆中间步骤 → 检查和比较路径 → 用更多推理时计算换更稳答案
```
