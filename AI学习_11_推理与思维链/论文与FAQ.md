# 论文与 FAQ

这个模块的论文主线是：如何用中间步骤、多路径和额外计算提升复杂任务表现。

## 推荐了解的主题

| 主题 | 可以关注什么 |
|------|--------------|
| Chain-of-Thought | 为什么分步推理能提升表现 |
| Self-Consistency | 多路径投票如何提高稳定性 |
| Tree of Thoughts | 搜索式推理如何展开候选 |
| Verifier / Process Supervision | 检查答案和检查过程有什么区别 |
| Test-time Compute | 推理时多花计算如何换质量 |

## FAQ

### 思维链是不是模型真实思考过程？

不一定。它是模型生成的中间文本或内部推理辅助，不能简单等同于人类思考。

### 分步回答一定更好吗？

不一定。简单问题分步会啰嗦，复杂问题分步更有帮助。

### 推理模型为什么更慢？

因为它在回答时投入了更多计算，用来生成、比较或检查中间过程。

### 高级推理能消除幻觉吗？

不能。它能减少某些推理错误，但事实问题仍需要证据和工具验证。
